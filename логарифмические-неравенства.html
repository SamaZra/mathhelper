Логарифмические неравенства

📖 Что такое логарифмическое неравенство

Логарифмическое неравенство — неравенство, в котором переменная содержится под знаком логарифма.

$\log_2 x > 3$
$\log_{0{,}5}(x+1) \leq 2$
$\log_3(x-1) < \log_3(2x+3)$
$\log_2^2 x - 3\log_2 x + 2 \geq 0$
💡 Ключевой принцип:
Если основание $a > 1$ — знак сохраняется
Если $0 < a < 1$ — знак меняется
И всегда проверяем ОДЗ: аргумент логарифма $> 0$!
📐 Основание больше 1 $a > 1$
Если $a > 1$, то $\log_a f(x) > \log_a g(x) \Leftrightarrow f(x) > g(x)$
знак неравенства сохраняется
Пример 1. $\log_2(x-1) > 3$

Шаг 1. ОДЗ. $x-1 > 0 \Rightarrow x > 1$.

Шаг 2. Представим $3$ как логарифм. $3 = \log_2 8$. Неравенство: $\log_2(x-1) > \log_2 8$.
Основание $2 > 1$ — знак сохраняется.

Шаг 3. $x-1 > 8 \Rightarrow x > 9$.
С учётом ОДЗ: $x > 9$.

Ответ: $x \in (9; +\infty)$.
Пример 2. $\log_3(x-1) < \log_3(2x+3)$

Шаг 1. ОДЗ. $x-1 > 0 \Rightarrow x > 1$
$2x+3 > 0 \Rightarrow x > -1{,}5$
Общая ОДЗ: $x > 1$.

Шаг 2. Основание $3 > 1$ — знак сохраняется.
$x-1 < 2x+3 \Rightarrow -x < 4 \Rightarrow x > -4$.

Шаг 3. Пересекаем с ОДЗ. $x > 1$.

Ответ: $x \in (1; +\infty)$.
📐 Основание от 0 до 1 $0 < a < 1$
Если $0 < a < 1$, то $\log_a f(x) > \log_a g(x) \Leftrightarrow f(x) < g(x)$
знак неравенства меняется на противоположный
Пример 3. $\log_{0{,}5}(x+1) \leq 2$

Шаг 1. ОДЗ. $x+1 > 0 \Rightarrow x > -1$.

Шаг 2. $2 = \log_{0{,}5}(0{,}25)$.
$\log_{0{,}5}(x+1) \leq \log_{0{,}5}(0{,}25)$.
Основание $0{,}5 \in (0; 1)$ — знак меняется!

Шаг 3. $x+1 \geq 0{,}25 \Rightarrow x \geq -0{,}75$.
С учётом ОДЗ: $x \geq -0{,}75$.

Ответ: $x \in [-0{,}75; +\infty)$.
Пример 4. $\log_{\frac{1}{3}}(2x-1) > \log_{\frac{1}{3}}(x+3)$

Шаг 1. ОДЗ. $2x-1 > 0 \Rightarrow x > 0{,}5$
$x+3 > 0 \Rightarrow x > -3$
Общая ОДЗ: $x > 0{,}5$.

Шаг 2. Основание $\dfrac{1}{3} \in (0; 1)$ — знак меняется!
$2x-1 < x+3 \Rightarrow x < 4$.

Шаг 3. Пересекаем с ОДЗ. $0{,}5 < x < 4$.

Ответ: $x \in (0{,}5; 4)$.
📐 Метод замены переменной сведение к квадратному неравенству
Пример 5. $\log_2^2 x - 3\log_2 x + 2 \geq 0$

Шаг 1. ОДЗ. $x > 0$.

Шаг 2. Замена $t = \log_2 x$. $t^2 - 3t + 2 \geq 0$.
Корни: $t_1 = 1$, $t_2 = 2$. Парабола ветвями вверх: $t \leq 1$ или $t \geq 2$.

Шаг 3. Возврат. $\log_2 x \leq 1 \Rightarrow 0 < x \leq 2$ (основание $2 > 1$ — знак сохраняется).
$\log_2 x \geq 2 \Rightarrow x \geq 4$.

Ответ: $x \in (0; 2] \cup [4; +\infty)$.
Пример 6. $\log_{0{,}5}^2 x + \log_{0{,}5} x - 2 \leq 0$

Шаг 1. ОДЗ. $x > 0$. Замена $t = \log_{0{,}5} x$.

Шаг 2. $t^2 + t - 2 \leq 0$. Корни: $t_1 = -2$, $t_2 = 1$. Решение: $-2 \leq t \leq 1$.

Шаг 3. Возврат. $-2 \leq \log_{0{,}5} x \leq 1$.
$\log_{0{,}5} x \geq -2 = \log_{0{,}5} 4$ — знак меняется: $x \leq 4$.
$\log_{0{,}5} x \leq 1 = \log_{0{,}5} 0{,}5$ — знак меняется: $x \geq 0{,}5$.
Пересечение: $0{,}5 \leq x \leq 4$.

Ответ: $x \in [0{,}5; 4]$.
📐 Смешанные неравенства ★ логарифмы с разными основаниями
Пример 7. $\log_2 x + \log_4 x < 3$

Шаг 1. ОДЗ. $x > 0$.

Шаг 2. Приводим к основанию $2$. $\log_4 x = \dfrac{\log_2 x}{2}$.
$\log_2 x + \dfrac{1}{2}\log_2 x < 3 \Rightarrow \dfrac{3}{2}\log_2 x < 3 \Rightarrow \log_2 x < 2$.

Шаг 3. Основание $2 > 1$ — знак сохраняется: $x < 4$.
С учётом ОДЗ: $0 < x < 4$.

Ответ: $x \in (0; 4)$.
📊 Сводная таблица
1. $\log_a f(x) > \log_a g(x)$, $a > 1$ — знак сохраняется: $f(x) > g(x)$

2. $\log_a f(x) > \log_a g(x)$, $0 < a < 1$ — знак меняется: $f(x) < g(x)$

3. Всегда проверяем ОДЗ: $f(x) > 0$, $g(x) > 0$

4. $\log_a^2 x + b\log_a x + c \geq 0$ — замена $t = \log_a x$

5. Разные основания — приводим к одному основанию
