Тригонометрические уравнения, сводящиеся к алгебраическим

📖 Что такое уравнения, сводящиеся к алгебраическим

Это уравнения, которые с помощью замены переменной $t = \sin x$, $t = \cos x$, $t = \operatorname{tg} x$ или $t = \operatorname{ctg} x$ сводятся к квадратным, кубическим или другим алгебраическим уравнениям.

$\sin^2 x + \sin x - 2 = 0$
$2\cos^2 x - 3\cos x + 1 = 0$
$\operatorname{tg}^2 x - 4\operatorname{tg} x + 3 = 0$
$\sin^4 x - \cos^4 x = 0$
💡 Ключевая идея: Замена тригонометрической функции на переменную $t$ превращает сложное тригонометрическое уравнение в знакомое алгебраическое. После нахождения корней $t$ возвращаемся к тригонометрическим уравнениям.
📐 Замена $t = \sin x$ или $t = \cos x$ квадратные уравнения относительно синуса или косинуса
Пример 1. $2\sin^2 x - 3\sin x + 1 = 0$

Шаг 1. Замена. Пусть $t = \sin x$, $t \in [-1; 1]$.
$2t^2 - 3t + 1 = 0$.

Шаг 2. Решаем квадратное уравнение. $D = 9 - 8 = 1$, $t_1 = 1$, $t_2 = \dfrac{1}{2}$.

Шаг 3. Обратная замена. $\sin x = 1 \Rightarrow x = \dfrac{\pi}{2} + 2\pi n$, $n \in \mathbb{Z}$.
$\sin x = \dfrac{1}{2} \Rightarrow x = \dfrac{\pi}{6} + 2\pi n$ или $x = \dfrac{5\pi}{6} + 2\pi n$, $n \in \mathbb{Z}$.

Ответ: $x = \dfrac{\pi}{2} + 2\pi n$, $x = \dfrac{\pi}{6} + 2\pi n$, $x = \dfrac{5\pi}{6} + 2\pi n$, $n \in \mathbb{Z}$.
📐 Замена $t = \operatorname{tg} x$ или $t = \operatorname{ctg} x$ уравнения с тангенсом и котангенсом
Пример 2. $\operatorname{tg}^2 x - 4\operatorname{tg} x + 3 = 0$

Шаг 1. Замена. Пусть $t = \operatorname{tg} x$, $t \in \mathbb{R}$.
$t^2 - 4t + 3 = 0$.

Шаг 2. Корни: $t_1 = 1$, $t_2 = 3$.

Шаг 3. Обратная замена. $\operatorname{tg} x = 1 \Rightarrow x = \dfrac{\pi}{4} + \pi n$, $n \in \mathbb{Z}$.
$\operatorname{tg} x = 3 \Rightarrow x = \operatorname{arctg} 3 + \pi n$, $n \in \mathbb{Z}$.

Ответ: $x = \dfrac{\pi}{4} + \pi n$, $x = \operatorname{arctg} 3 + \pi n$, $n \in \mathbb{Z}$.
📐 Замена $t = \sin x \pm \cos x$ уравнения, содержащие $\sin x + \cos x$ и $\sin x \cos x$

Используем соотношение: $(\sin x + \cos x)^2 = 1 + 2\sin x \cos x$.

Пример 3. $\sin x + \cos x + \sin x \cos x = 1$

Шаг 1. Замена. Пусть $t = \sin x + \cos x$, $t \in [-\sqrt{2}; \sqrt{2}]$.
Тогда $\sin x \cos x = \dfrac{t^2 - 1}{2}$.

Шаг 2. Подставляем. $t + \dfrac{t^2 - 1}{2} = 1 \Rightarrow 2t + t^2 - 1 = 2 \Rightarrow t^2 + 2t - 3 = 0$.
Корни: $t_1 = 1$, $t_2 = -3$ (не подходит, так как $t \in [-\sqrt{2}; \sqrt{2}]$).

Шаг 3. Обратная замена. $\sin x + \cos x = 1 \Rightarrow \sqrt{2}\sin\left(x + \dfrac{\pi}{4}\right) = 1 \Rightarrow \sin\left(x + \dfrac{\pi}{4}\right) = \dfrac{\sqrt{2}}{2}$.
$x + \dfrac{\pi}{4} = \dfrac{\pi}{4} + 2\pi n \Rightarrow x = 2\pi n$
$x + \dfrac{\pi}{4} = \dfrac{3\pi}{4} + 2\pi n \Rightarrow x = \dfrac{\pi}{2} + 2\pi n$

Ответ: $x = 2\pi n$, $x = \dfrac{\pi}{2} + 2\pi n$, $n \in \mathbb{Z}$.
📐 Однородные уравнения $a\sin x + b\cos x = 0$ и $a\sin^2 x + b\sin x \cos x + c\cos^2 x = 0$

Однородное уравнение 1-й степени: $a\sin x + b\cos x = 0$. Делим на $\cos x \neq 0$.

Пример 4. $\sin x + \cos x = 0$

Шаг 1. Делим на $\cos x$ (проверяем, что $\cos x = 0$ не является решением).
$\operatorname{tg} x + 1 = 0 \Rightarrow \operatorname{tg} x = -1$.

Ответ: $x = -\dfrac{\pi}{4} + \pi n$, $n \in \mathbb{Z}$.

Однородное уравнение 2-й степени: $a\sin^2 x + b\sin x \cos x + c\cos^2 x = 0$. Делим на $\cos^2 x \neq 0$.

Пример 5. $\sin^2 x - 3\sin x \cos x + 2\cos^2 x = 0$

Шаг 1. Делим на $\cos^2 x$:
$\operatorname{tg}^2 x - 3\operatorname{tg} x + 2 = 0$.

Шаг 2. $t = \operatorname{tg} x$, $t^2 - 3t + 2 = 0 \Rightarrow t_1 = 1$, $t_2 = 2$.

Шаг 3. Обратная замена. $\operatorname{tg} x = 1 \Rightarrow x = \dfrac{\pi}{4} + \pi n$
$\operatorname{tg} x = 2 \Rightarrow x = \operatorname{arctg} 2 + \pi n$

Ответ: $x = \dfrac{\pi}{4} + \pi n$, $x = \operatorname{arctg} 2 + \pi n$, $n \in \mathbb{Z}$.
📐 Уравнения вида $a\sin x + b\cos x = c$ метод введения вспомогательного аргумента
Метод:
$a\sin x + b\cos x = R\sin(x + \varphi)$, где $R = \sqrt{a^2 + b^2}$, $\cos\varphi = \dfrac{a}{R}$, $\sin\varphi = \dfrac{b}{R}$.
Пример 6. $\sin x + \sqrt{3}\cos x = 1$

Шаг 1. $R = \sqrt{1 + 3} = 2$.
$\sin x + \sqrt{3}\cos x = 2\sin\left(x + \dfrac{\pi}{3}\right)$.

Шаг 2. $2\sin\left(x + \dfrac{\pi}{3}\right) = 1 \Rightarrow \sin\left(x + \dfrac{\pi}{3}\right) = \dfrac{1}{2}$.

Шаг 3. $x + \dfrac{\pi}{3} = \dfrac{\pi}{6} + 2\pi n \Rightarrow x = -\dfrac{\pi}{6} + 2\pi n$
$x + \dfrac{\pi}{3} = \dfrac{5\pi}{6} + 2\pi n \Rightarrow x = \dfrac{\pi}{2} + 2\pi n$

Ответ: $x = -\dfrac{\pi}{6} + 2\pi n$, $x = \dfrac{\pi}{2} + 2\pi n$, $n \in \mathbb{Z}$.
📐 Универсальная подстановка $t = \operatorname{tg}\frac{x}{2}$

Универсальная подстановка позволяет свести любое уравнение с рациональными комбинациями $\sin x$ и $\cos x$ к дробно-рациональному уравнению относительно $t$.

Формулы замены ($t = \operatorname{tg}\frac{x}{2}$):
$\sin x = \dfrac{2t}{1 + t^2}$
$\cos x = \dfrac{1 - t^2}{1 + t^2}$
$\operatorname{tg} x = \dfrac{2t}{1 - t^2}$

Важно: $t \in \mathbb{R}$, кроме случая $x = \pi + 2\pi n$.
⚠️ ВАЖНО! Проверка потери корня.

При замене $t = \operatorname{tg}\frac{x}{2}$ мы теряем точки, где $\operatorname{tg}\frac{x}{2}$ не определён, то есть $x = \pi + 2\pi n$.

Алгоритм проверки:
  1. Подставьте $x = \pi + 2\pi n$ в исходное уравнение.
  2. Если равенство выполняется — это корень. Запомните его и добавьте в финальный ответ.
  3. Если не выполняется — потери нет, спокойно делайте замену.
Запомните: Всегда проверяйте $x = \pi + 2\pi n$ перед заменой!
Пример 7 (корень не теряется). Решить уравнение $6\cos x - 5\sin x = 7$

Шаг 1. Проверяем потерю корня. Подставляем $x = \pi + 2\pi n$:
$\cos(\pi + 2\pi n) = -1$, $\sin(\pi + 2\pi n) = 0$.
$6 \cdot (-1) - 5 \cdot 0 = -6 \neq 7$.
➡️ Корень не подходит, потери нет.

Шаг 2. Замена $t = \operatorname{tg}\frac{x}{2}$. $\sin x = \dfrac{2t}{1+t^2}$, $\cos x = \dfrac{1-t^2}{1+t^2}$.

Шаг 3. Подставляем в уравнение. $6 \cdot \dfrac{1-t^2}{1+t^2} - 5 \cdot \dfrac{2t}{1+t^2} = 7$
$6(1-t^2) - 10t = 7(1+t^2)$
$6 - 6t^2 - 10t = 7 + 7t^2$
$-13t^2 - 10t - 1 = 0$
$13t^2 + 10t + 1 = 0$

Шаг 4. Решаем квадратное уравнение. $D = 100 - 52 = 48 = 16 \cdot 3$
$t_{1,2} = \dfrac{-10 \pm 4\sqrt{3}}{26} = \dfrac{-5 \pm 2\sqrt{3}}{13}$

Шаг 5. Обратная замена. $\operatorname{tg}\frac{x}{2} = \dfrac{-5 \pm 2\sqrt{3}}{13}$
$\dfrac{x}{2} = \operatorname{arctg}\left(\dfrac{-5 \pm 2\sqrt{3}}{13}\right) + \pi n$
$x = 2\operatorname{arctg}\left(\dfrac{-5 \pm 2\sqrt{3}}{13}\right) + 2\pi n$

Ответ: $x = 2\operatorname{arctg}\left(\dfrac{-5 \pm 2\sqrt{3}}{13}\right) + 2\pi n$, $n \in \mathbb{Z}$.
Пример 8 (корень теряется!). Решить уравнение $\sin x = 0$

Шаг 1. Проверяем потерю корня. Подставляем $x = \pi + 2\pi n$:
$\sin(\pi + 2\pi n) = 0$.
➡️ Это корень! Запоминаем: $x = \pi + 2\pi n$.

Шаг 2. Замена $t = \operatorname{tg}\frac{x}{2}$. $\sin x = \dfrac{2t}{1+t^2}$.

Шаг 3. Подставляем. $\dfrac{2t}{1+t^2} = 0 \Rightarrow 2t = 0 \Rightarrow t = 0$.

Шаг 4. Обратная замена. $\operatorname{tg}\frac{x}{2} = 0 \Rightarrow \dfrac{x}{2} = \pi n \Rightarrow x = 2\pi n$.

Шаг 5. Объединяем ответы. Решения через замену: $x = 2\pi n$.
Найденный потерянный корень: $x = \pi + 2\pi n$.
Объединяем: $x = \pi n$ (все целые кратные $\pi$).

Ответ: $x = \pi n$, $n \in \mathbb{Z}$.

📌 Вывод: Без проверки мы бы потеряли корень $x = \pi + 2\pi n$ и ошибочно записали ответ только как $x = 2\pi n$.

$t \in [-1; 1]$. После решения $t$ — решаем простейшие уравнения.

2. Замена $t = \operatorname{tg} x$ или $t = \operatorname{ctg} x$
$t \in \mathbb{R}$. Проверяем, не теряем ли корни при делении.

3. Замена $t = \sin x \pm \cos x$
$t \in [-\sqrt{2}; \sqrt{2}]$. $\sin x \cos x = \dfrac{t^2 \mp 1}{2}$.

4. Однородные
$a\sin x + b\cos x = 0$ — делим на $\cos x$.
$a\sin^2 x + b\sin x \cos x + c\cos^2 x = 0$ — делим на $\cos^2 x$.

5. $a\sin x + b\cos x = c$
Вводим вспомогательный аргумент: $R = \sqrt{a^2 + b^2}$.

6. Универсальная подстановка $t = \operatorname{tg}\frac{x}{2}$
$\sin x = \dfrac{2t}{1+t^2}$, $\cos x = \dfrac{1-t^2}{1+t^2}$.
⚠️ Перед заменой проверяем $x = \pi + 2\pi n$!
