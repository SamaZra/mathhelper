Сумма и разность углов

\(\sin(\alpha \pm \beta) = \sin\alpha\cos\beta \pm \cos\alpha\sin\beta\)

\(\cos(\alpha \pm \beta) = \cos\alpha\cos\beta \mp \sin\alpha\sin\beta\)

\(\tg(\alpha + \beta) = \dfrac{\tg\alpha + \tg\beta}{1 - \tg\alpha\tg\beta}\)

\(\tg(\alpha - \beta) = \dfrac{\tg\alpha - \tg\beta}{1 + \tg\alpha\tg\beta}\)

📝 Примеры применения

Пример 1. Вычислить \(\sin 75^\circ\).

Решение:

Представим \(75^\circ = 45^\circ + 30^\circ\). Используем формулу \(\sin(\alpha + \beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta\):

\(\sin 75^\circ = \sin 45^\circ\cos 30^\circ + \cos 45^\circ\sin 30^\circ =\)

\(= \dfrac{\sqrt{2}}{2} \cdot \dfrac{\sqrt{3}}{2} + \dfrac{\sqrt{2}}{2} \cdot \dfrac{1}{2} =\)

\(= \dfrac{\sqrt{6}}{4} + \dfrac{\sqrt{2}}{4} = \dfrac{\sqrt{6} + \sqrt{2}}{4}\).

Ответ: \(\dfrac{\sqrt{6} + \sqrt{2}}{4}\).

Пример 2. Упростить: \(\cos 3\alpha\cos\alpha + \sin 3\alpha\sin\alpha\).

Решение:

Используем формулу \(\cos(\alpha - \beta) = \cos\alpha\cos\beta + \sin\alpha\sin\beta\):

\(\cos 3\alpha\cos\alpha + \sin 3\alpha\sin\alpha = \cos(3\alpha - \alpha) =\)

\(= \cos 2\alpha\).

Ответ: \(\cos 2\alpha\).

Пример 3. Найти \(\tg 15^\circ\).

Решение:

Представим \(15^\circ = 45^\circ - 30^\circ\). Используем формулу \(\tg(\alpha - \beta) = \dfrac{\tg\alpha - \tg\beta}{1 + \tg\alpha\tg\beta}\):

\(\tg 15^\circ = \dfrac{\tg 45^\circ - \tg 30^\circ}{1 + \tg 45^\circ\tg 30^\circ} =\)

\(= \dfrac{1 - \frac{\sqrt{3}}{3}}{1 + 1\cdot\frac{\sqrt{3}}{3}} = 2 - \sqrt{3}\).

Ответ: \(2 - \sqrt{3}\).
