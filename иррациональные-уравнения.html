Иррациональные уравнения

?? Что такое иррациональное уравнение

Иррациональное уравнение — уравнение, в котором переменная содержится под знаком корня (радикала).

$\sqrt{x+1} = 3$
$\sqrt[3]{2x-1} = \sqrt{x}$
$\sqrt{x+5} + \sqrt{x} = 5$
?? Важно: При решении необходимо учитывать ОДЗ — подкоренное выражение корня чётной степени должно быть $\geq 0$!
?? Метод возведения в степень основной алгоритм решения
Алгоритм:
1. Найти ОДЗ
2. Возвести обе части в степень
3. Решить полученное уравнение
4. Проверить корни подстановкой в исходное уравнение
Пример 1: Решить уравнение $\sqrt{x+2} = 4$.

Шаг 1. Находим ОДЗ. Подкоренное выражение должно быть неотрицательным:
$x + 2 \geq 0$
$x \geq -2$

Шаг 2. Возводим обе части в квадрат. $(\sqrt{x+2})^2 = 4^2$
$x + 2 = 16$

Шаг 3. Решаем полученное уравнение. $x = 16 - 2$
$x = 14$

Шаг 4. Проверка. Подставляем $x = 14$ в исходное уравнение:
$\sqrt{14 + 2} = \sqrt{16} = 4$
$4 = 4$ — верно.
Также проверяем ОДЗ: $14 \geq -2$ — подходит.

Ответ: $x = 14$.
Пример 2: Решить уравнение $\sqrt{x+3} = x-3$.

Шаг 1. Находим ОДЗ. Подкоренное выражение: $x + 3 \geq 0 \Rightarrow x \geq -3$.
Правая часть (значение корня): $x - 3 \geq 0 \Rightarrow x \geq 3$.
Общая ОДЗ: $x \geq 3$.

Шаг 2. Возводим обе части в квадрат. $(\sqrt{x+3})^2 = (x-3)^2$
$x + 3 = x^2 - 6x + 9$

Шаг 3. Решаем квадратное уравнение. $x^2 - 6x + 9 - x - 3 = 0$
$x^2 - 7x + 6 = 0$
Находим дискриминант: $D = (-7)^2 - 4 \cdot 1 \cdot 6 = 49 - 24 = 25$
$\sqrt{D} = 5$
$x_1 = \dfrac{7 + 5}{2} = 6$
$x_2 = \dfrac{7 - 5}{2} = 1$

Шаг 4. Проверка корней с учётом ОДЗ. Проверяем $x = 6$:
$\sqrt{6 + 3} = \sqrt{9} = 3$
Правая часть: $6 - 3 = 3$
$3 = 3$ — верно. ОДЗ: $6 \geq 3$ — подходит.

Проверяем $x = 1$:
ОДЗ: $1 \geq 3$ — не выполняется.
Корень $x = 1$ не удовлетворяет ОДЗ.

Ответ: $x = 6$.
?? Уравнения с кубическими корнями корни нечётной степени — ОДЗ не нужен

Для корней нечётной степени ОДЗ не требуется — подкоренное выражение может быть любым действительным числом.

Пример 3: Решить уравнение $\sqrt[3]{2x-1} = 3$.

Шаг 1. ОДЗ. Корень кубический определён при любом $x$, поэтому ОДЗ: $x \in \mathbb{R}$.

Шаг 2. Возводим обе части в куб. $(\sqrt[3]{2x-1})^3 = 3^3$
$2x - 1 = 27$

Шаг 3. Решаем линейное уравнение. $2x = 27 + 1$
$2x = 28$
$x = 14$

Шаг 4. Проверка. Подставляем $x = 14$:
$\sqrt[3]{2 \cdot 14 - 1} = \sqrt[3]{28 - 1} = \sqrt[3]{27} = 3$
$3 = 3$ — верно.

Ответ: $x = 14$.
Пример 4: Решить уравнение $\sqrt[3]{x+1} = \sqrt[3]{2x-3}$.

Шаг 1. ОДЗ. Оба корня кубические, определены при любых $x$. ОДЗ: $x \in \mathbb{R}$.

Шаг 2. Возводим обе части в куб. $(\sqrt[3]{x+1})^3 = (\sqrt[3]{2x-3})^3$
$x + 1 = 2x - 3$

Шаг 3. Решаем линейное уравнение. $x - 2x = -3 - 1$
$-x = -4$
$x = 4$

Шаг 4. Проверка. Левая часть: $\sqrt[3]{4 + 1} = \sqrt[3]{5}$
Правая часть: $\sqrt[3]{2 \cdot 4 - 3} = \sqrt[3]{8 - 3} = \sqrt[3]{5}$
$\sqrt[3]{5} = \sqrt[3]{5}$ — верно.

Ответ: $x = 4$.
?? Метод замены переменной сведение к квадратному уравнению
Пример 5: Решить уравнение $\sqrt{x} + \sqrt[4]{x} = 6$.

Шаг 1. ОДЗ. $\sqrt{x}$ определён при $x \geq 0$, $\sqrt[4]{x}$ также при $x \geq 0$.
ОДЗ: $x \geq 0$.

Шаг 2. Вводим замену переменной. Пусть $t = \sqrt[4]{x}$, тогда $t \geq 0$.
Выразим $\sqrt{x}$ через $t$: $\sqrt{x} = (\sqrt[4]{x})^2 = t^2$.
Уравнение принимает вид:
$t^2 + t = 6$
$t^2 + t - 6 = 0$

Шаг 3. Решаем квадратное уравнение относительно $t$. $D = 1^2 - 4 \cdot 1 \cdot (-6) = 1 + 24 = 25$
$\sqrt{D} = 5$
$t_1 = \dfrac{-1 + 5}{2} = 2$
$t_2 = \dfrac{-1 - 5}{2} = -3$

Шаг 4. Отбираем корни с учётом условия $t \geq 0$. $t = 2$ — подходит.
$t = -3$ — не подходит (корень четвёртой степени не может быть отрицательным).

Шаг 5. Возвращаемся к переменной $x$. $\sqrt[4]{x} = 2$
Возводим в четвёртую степень:
$x = 2^4 = 16$

Шаг 6. Проверка. $\sqrt{16} + \sqrt[4]{16} = 4 + 2 = 6$
$6 = 6$ — верно.

Ответ: $x = 16$.
?? Метод уединения радикала уравнения с несколькими корнями

Если в уравнении несколько корней, нужно уединить один из них (перенести всё остальное в другую часть) и возвести в степень.

Пример 6: Решить уравнение $\sqrt{x+5} + \sqrt{x} = 5$.

Шаг 1. Находим ОДЗ. $\sqrt{x+5}$: $x + 5 \geq 0 \Rightarrow x \geq -5$
$\sqrt{x}$: $x \geq 0$
Общая ОДЗ: $x \geq 0$.

Шаг 2. Уединяем радикал. Переносим $\sqrt{x}$ в правую часть:
$\sqrt{x+5} = 5 - \sqrt{x}$
Заметим, что правая часть должна быть неотрицательна:
$5 - \sqrt{x} \geq 0 \Rightarrow \sqrt{x} \leq 5 \Rightarrow x \leq 25$

Шаг 3. Возводим обе части в квадрат. $(\sqrt{x+5})^2 = (5 - \sqrt{x})^2$
$x + 5 = 25 - 10\sqrt{x} + x$

Шаг 4. Приводим подобные. $x + 5 - 25 - x = -10\sqrt{x}$
$-20 = -10\sqrt{x}$
$10\sqrt{x} = 20$
$\sqrt{x} = 2$

Шаг 5. Возводим в квадрат ещё раз. $x = 4$

Шаг 6. Проверка. $\sqrt{4 + 5} + \sqrt{4} = \sqrt{9} + 2 = 3 + 2 = 5$
$5 = 5$ — верно.
Проверяем ОДЗ: $x = 4$, $0 \leq 4 \leq 25$ — подходит.

Ответ: $x = 4$.
?? Повторное возведение в степень если радикал не исчез после первого шага

Иногда после первого возведения в квадрат корень остаётся — тогда возводим ещё раз.

Пример 7: Решить уравнение $\sqrt{x+3} + \sqrt{x-2} = 5$.

Шаг 1. Находим ОДЗ. $\sqrt{x+3}$: $x + 3 \geq 0 \Rightarrow x \geq -3$
$\sqrt{x-2}$: $x - 2 \geq 0 \Rightarrow x \geq 2$
Общая ОДЗ: $x \geq 2$.

Шаг 2. Уединяем первый радикал. $\sqrt{x+3} = 5 - \sqrt{x-2}$
Правая часть: $5 - \sqrt{x-2} \geq 0 \Rightarrow \sqrt{x-2} \leq 5 \Rightarrow x-2 \leq 25 \Rightarrow x \leq 27$

Шаг 3. Возводим в квадрат первый раз. $(\sqrt{x+3})^2 = (5 - \sqrt{x-2})^2$
$x + 3 = 25 - 10\sqrt{x-2} + (x - 2)$
$x + 3 = 25 - 10\sqrt{x-2} + x - 2$
$x + 3 = x + 23 - 10\sqrt{x-2}$

Шаг 4. Приводим подобные. $x + 3 - x - 23 = -10\sqrt{x-2}$
$-20 = -10\sqrt{x-2}$
$10\sqrt{x-2} = 20$
$\sqrt{x-2} = 2$

Шаг 5. Возводим в квадрат второй раз. $x - 2 = 4$
$x = 6$

Шаг 6. Проверка. Подставляем $x = 6$ в исходное уравнение:
$\sqrt{6 + 3} + \sqrt{6 - 2} = \sqrt{9} + \sqrt{4} = 3 + 2 = 5$
$5 = 5$ — верно.
Проверяем ОДЗ: $2 \leq 6 \leq 27$ — подходит.

Ответ: $x = 6$.
?? Системы иррациональных уравнений замена переменных и симметричные системы
Пример 8: Решить систему уравнений $\begin{cases} \sqrt{x} + \sqrt{y} = 5 \\ x + y = 13 \end{cases}$.

Шаг 1. ОДЗ. $\sqrt{x}$ определён при $x \geq 0$, $\sqrt{y}$ — при $y \geq 0$.
ОДЗ: $x \geq 0$, $y \geq 0$.

Шаг 2. Вводим замену переменных. Пусть $a = \sqrt{x} \geq 0$, $b = \sqrt{y} \geq 0$.
Тогда $x = a^2$, $y = b^2$.
Система принимает вид:
$\begin{cases} a + b = 5 \\ a^2 + b^2 = 13 \end{cases}$

Шаг 3. Выражаем $a^2 + b^2$ через сумму и произведение. $a^2 + b^2 = (a+b)^2 - 2ab$
$13 = 5^2 - 2ab$
$13 = 25 - 2ab$
$2ab = 25 - 13 = 12$
$ab = 6$

Шаг 4. Составляем квадратное уравнение. Числа $a$ и $b$ — корни уравнения $t^2 - 5t + 6 = 0$ (по теореме Виета: сумма корней $5$, произведение $6$).
Решаем: $D = 25 - 24 = 1$
$t_1 = \dfrac{5 + 1}{2} = 3$
$t_2 = \dfrac{5 - 1}{2} = 2$

Шаг 5. Находим $a$ и $b$. Возможны два случая:
Случай 1: $a = 3$, $b = 2$
Случай 2: $a = 2$, $b = 3$
Оба подходят, так как $a,b \geq 0$.

Шаг 6. Возвращаемся к $x$ и $y$. Случай 1: $\sqrt{x} = 3 \Rightarrow x = 9$, $\sqrt{y} = 2 \Rightarrow y = 4$.
Случай 2: $\sqrt{x} = 2 \Rightarrow x = 4$, $\sqrt{y} = 3 \Rightarrow y = 9$.

Шаг 7. Проверка. Для $(9; 4)$: $\sqrt{9} + \sqrt{4} = 3 + 2 = 5$, $9 + 4 = 13$ — верно.
Для $(4; 9)$: $\sqrt{4} + \sqrt{9} = 2 + 3 = 5$, $4 + 9 = 13$ — верно.

Ответ: $(4; 9)$ и $(9; 4)$.
?? Задачи с параметром ? анализ количества корней
Пример 9: При каких значениях параметра $a$ уравнение $\sqrt{x+a} = x$ имеет ровно один корень?

Шаг 1. ОДЗ. Корень квадратный: $x + a \geq 0 \Rightarrow x \geq -a$.
Правая часть (значение корня): $x \geq 0$.
Общая ОДЗ: $x \geq \max(0, -a)$.

Шаг 2. Возводим в квадрат. $x + a = x^2$
$x^2 - x - a = 0$

Шаг 3. Анализируем квадратное уравнение. Дискриминант: $D = 1 + 4a$.
Уравнение имеет корни, если $D \geq 0$, то есть $a \geq -\dfrac{1}{4}$.
Корни: $x_{1,2} = \dfrac{1 \pm \sqrt{1 + 4a}}{2}$.

Шаг 4. Проверяем ОДЗ для каждого корня. Оба корня должны удовлетворять $x \geq 0$ и $x \geq -a$.

Случай $a < 0$: $-a > 0$, ОДЗ: $x \geq -a$.
Меньший корень: $x_2 = \dfrac{1 - \sqrt{1+4a}}{2}$.
При $a < 0$: $\sqrt{1+4a} < 1$, значит $x_2 > 0$.
Проверяем $x_2 \geq -a$: это выполняется не всегда. Анализ показывает, что при $a < 0$ подходит только больший корень, если $a > -2$.

Случай $a = 0$: $x^2 - x = 0$, корни $0$ и $1$. Оба подходят — два корня.

Случай $a > 0$: $-a < 0$, ОДЗ: $x \geq 0$.
Меньший корень $x_2 < 0$ (не подходит). Больший корень $x_1 > 0$ (подходит). Итого один корень.

Случай $a = -\dfrac{1}{4}$: $D = 0$, единственный корень $x = \dfrac{1}{2}$. Проверяем ОДЗ: подходит.

Шаг 5. Итоговый анализ. Один корень будет при $a = -\dfrac{1}{4}$ и при $a > 0$.
При $a \in (-\dfrac{1}{4}; 0]$ — два корня.
При $a < -\dfrac{1}{4}$ — корней нет.

Ответ: $a = -\dfrac{1}{4}$ или $a > 0$.
Пример 10 (ЕГЭ): Найти все значения $a$, при которых уравнение $\sqrt{15x^2+6ax+9} = x^2+ax+3$ имеет ровно три различных корня.

Шаг 1. ОДЗ. Правая часть (значение корня): $x^2 + ax + 3 \geq 0$.
Дискриминант правой части: $D = a^2 - 12$.
Если $|a| \leq 2\sqrt{3}$, то $x^2+ax+3 > 0$ при всех $x$.

Шаг 2. Возводим в квадрат. $15x^2 + 6ax + 9 = (x^2 + ax + 3)^2$
Раскрываем правую часть:
$(x^2 + ax + 3)^2 = x^4 + 2ax^3 + (a^2+6)x^2 + 6ax + 9$

Шаг 3. Приравниваем и приводим подобные. $15x^2 + 6ax + 9 = x^4 + 2ax^3 + (a^2+6)x^2 + 6ax + 9$
$0 = x^4 + 2ax^3 + (a^2+6-15)x^2 + (6a-6a)x + (9-9)$
$0 = x^4 + 2ax^3 + (a^2-9)x^2$
$x^2(x^2 + 2ax + a^2 - 9) = 0$

Шаг 4. Решаем уравнение. $x^2 = 0 \Rightarrow x = 0$ (кратный корень, считается за один).
$x^2 + 2ax + a^2 - 9 = 0$
$(x + a)^2 - 9 = 0$
$(x + a)^2 = 9$
$x + a = \pm 3$
$x = -a \pm 3$

Шаг 5. Анализируем количество корней. Имеем три кандидата: $x = 0$, $x = -a+3$, $x = -a-3$.
Чтобы было ровно три РАЗЛИЧНЫХ корня, все три должны быть различны и удовлетворять ОДЗ.

Условие $x=0$ совпадает с $x=-a+3$, если $-a+3=0 \Rightarrow a=3$.
Условие $x=0$ совпадает с $x=-a-3$, если $-a-3=0 \Rightarrow a=-3$.
При $a \neq \pm 3$ все три корня различны.

Шаг 6. Проверяем ОДЗ. Для $a = 3$: корни $0$, $0$, $-6$ — два различных (0 и -6), не подходит.
Для $a = -3$: корни $0$, $6$, $0$ — два различных (0 и 6), не подходит.
Для других $a$ нужно проверить ОДЗ. Анализ показывает, что при $|a| < \sqrt{12}$ все три корня удовлетворяют ОДЗ.

Ответ: $a \in (-2\sqrt{3}; 2\sqrt{3}) \setminus \{-3; 3\}$.
