Простейшие тригонометрические уравнения

📖 Арксинус, арккосинус, арктангенс, арккотангенс

Арксинус числа \(a \in [-1, 1]\) — это угол \(\alpha \in \left[-\frac{\pi}{2}, \frac{\pi}{2}\right]\), синус которого равен \(a\):

\(\arcsin a = \alpha \;\Leftrightarrow\; \sin\alpha = a,\; \alpha \in \left[-\frac{\pi}{2}, \frac{\pi}{2}\right]\).

Примеры: \(\arcsin 0 = 0,\; \arcsin\frac{1}{2} = \frac{\pi}{6},\; \arcsin 1 = \frac{\pi}{2}\).

Арккосинус числа \(a \in [-1, 1]\) — это угол \(\alpha \in [0, \pi]\), косинус которого равен \(a\):

\(\arccos a = \alpha \;\Leftrightarrow\; \cos\alpha = a,\; \alpha \in [0, \pi]\).

Примеры: \(\arccos 1 = 0,\; \arccos\frac{1}{2} = \frac{\pi}{3},\; \arccos 0 = \frac{\pi}{2}\).

Арктангенс числа \(a \in \mathbb{R}\) — это угол \(\alpha \in \left(-\frac{\pi}{2}, \frac{\pi}{2}\right)\), тангенс которого равен \(a\):

\(\arctg a = \alpha \;\Leftrightarrow\; \tg\alpha = a,\; \alpha \in \left(-\frac{\pi}{2}, \frac{\pi}{2}\right)\).

Примеры: \(\arctg 0 = 0,\; \arctg 1 = \frac{\pi}{4},\; \arctg\sqrt{3} = \frac{\pi}{3}\).

Арккотангенс числа \(a \in \mathbb{R}\) — это угол \(\alpha \in (0, \pi)\), котангенс которого равен \(a\):

\(\arcctg a = \alpha \;\Leftrightarrow\; \ctg\alpha = a,\; \alpha \in (0, \pi)\).

Примеры: \(\arcctg 1 = \frac{\pi}{4},\; \arcctg 0 = \frac{\pi}{2},\; \arcctg\sqrt{3} = \frac{\pi}{6}\).

sin x = a

\(\sin x = a,\; |a| \le 1\)

\(x = (-1)^n \arcsin a + \pi n,\; n \in \mathbb{Z}\)

Альтернативная форма:

\(\left[ \begin{array}{l} x = \arcsin a + 2\pi n \\ x = \pi - \arcsin a + 2\pi n \end{array} \right.\)

Частные случаи

sin x = 1

\(x = \dfrac{\pi}{2} + 2\pi n\)

sin x = 0

\(x = \pi n\)

sin x = -1

\(x = -\dfrac{\pi}{2} + 2\pi n\)

📝 Примеры с решением

Пример 1. Решить уравнение: \(\sin x = \dfrac{\sqrt{3}}{2}\).

Решение:

\(\sin x = \dfrac{\sqrt{3}}{2}\) — табличное значение, \(\arcsin\dfrac{\sqrt{3}}{2} = \dfrac{\pi}{3}\).

Используем формулу: \(x = (-1)^n\dfrac{\pi}{3} + \pi n,\; n \in \mathbb{Z}\).

Ответ: \(x = (-1)^n\dfrac{\pi}{3} + \pi n,\; n \in \mathbb{Z}\).

Пример 2. Решить уравнение: \(\sin x = -\dfrac{1}{2}\). Найти корни на отрезке \([0, 2\pi]\).

Решение:

\(\arcsin\left(-\dfrac{1}{2}\right) = -\dfrac{\pi}{6}\).

\(x = (-1)^n\left(-\dfrac{\pi}{6}\right) + \pi n\).

\(n = 1\): \(x = \dfrac{7\pi}{6}\); \(n = 2\): \(x = \dfrac{11\pi}{6}\).

Ответ: \(x = \dfrac{7\pi}{6},\; x = \dfrac{11\pi}{6}\).

cos x = a

\(\cos x = a,\; |a| \le 1\)

\(x = \pm \arccos a + 2\pi n,\; n \in \mathbb{Z}\)

Частные случаи

cos x = 1

\(x = 2\pi n\)

cos x = 0

\(x = \dfrac{\pi}{2} + \pi n\)

cos x = -1

\(x = \pi + 2\pi n\)

📝 Примеры с решением

Пример 1. Решить уравнение: \(\cos x = \dfrac{1}{2}\).

Решение:

\(\arccos\dfrac{1}{2} = \dfrac{\pi}{3}\).

\(x = \pm\dfrac{\pi}{3} + 2\pi n,\; n \in \mathbb{Z}\).

Ответ: \(x = \pm\dfrac{\pi}{3} + 2\pi n,\; n \in \mathbb{Z}\).

Пример 2. Решить уравнение: \(\cos x = -\dfrac{\sqrt{2}}{2}\). Найти корни на отрезке \([0, 2\pi]\).

Решение:

\(\arccos\left(-\dfrac{\sqrt{2}}{2}\right) = \dfrac{3\pi}{4}\).

\(x = \pm\dfrac{3\pi}{4} + 2\pi n\).

\(n = 0\): \(x = \dfrac{3\pi}{4},\; x = -\dfrac{3\pi}{4} \notin [0, 2\pi]\).

\(n = 1\): \(x = -\dfrac{3\pi}{4} + 2\pi = \dfrac{5\pi}{4} \in [0, 2\pi]\).

Ответ: \(x = \dfrac{3\pi}{4},\; x = \dfrac{5\pi}{4}\).

tg x = a

\(\tg x = a,\; a \in \mathbb{R}\)

\(x = \arctg a + \pi n,\; n \in \mathbb{Z}\)

tg x = 0

\(x = \pi n\)

📝 Примеры с решением

Пример 1. Решить уравнение: \(2\tg(3x - \frac{\pi}{4}) = 1\).

Решение:

\(\tg(3x - \frac{\pi}{4}) = \dfrac{1}{2}\).

\(3x - \dfrac{\pi}{4} = \arctg\dfrac{1}{2} + \pi n\).

\(3x = \arctg\dfrac{1}{2} + \dfrac{\pi}{4} + \pi n\).

\(x = \dfrac{1}{3}\left(\arctg\dfrac{1}{2} + \dfrac{\pi}{4} + \pi n\right),\; n \in \mathbb{Z}\).

Ответ: \(x = \dfrac{1}{3}\arctg\dfrac{1}{2} + \dfrac{\pi}{12} + \dfrac{\pi n}{3},\; n \in \mathbb{Z}\).

Пример 2. Решить уравнение: \(\tg^2 x - 3 = 0\). Найти корни на отрезке \([0, \pi]\).

Решение:

\(\tg^2 x = 3 \Rightarrow \tg x = \pm\sqrt{3}\).

\(\tg x = \sqrt{3} \Rightarrow x = \dfrac{\pi}{3} + \pi n\).

\(\tg x = -\sqrt{3} \Rightarrow x = -\dfrac{\pi}{3} + \pi n\).

На \([0, \pi]\): \(x = \dfrac{\pi}{3}\), \(x = \dfrac{2\pi}{3}\).

Ответ: \(x = \dfrac{\pi}{3},\; x = \dfrac{2\pi}{3}\).

ctg x = a

\(\ctg x = a,\; a \in \mathbb{R}\)

\(x = \arcctg a + \pi n,\; n \in \mathbb{Z}\)

ctg x = 0

\(x = \dfrac{\pi}{2} + \pi n\)

📝 Примеры с решением

Пример 1. Решить уравнение: \(3\ctg(2x + \frac{\pi}{6}) = \sqrt{3}\).

Решение:

\(\ctg(2x + \frac{\pi}{6}) = \dfrac{\sqrt{3}}{3}\).

\(2x + \dfrac{\pi}{6} = \arcctg\dfrac{\sqrt{3}}{3} + \pi n\).

\(\arcctg\dfrac{\sqrt{3}}{3} = \dfrac{\pi}{3}\).

\(2x = \dfrac{\pi}{3} - \dfrac{\pi}{6} + \pi n = \dfrac{\pi}{6} + \pi n\).

\(x = \dfrac{\pi}{12} + \dfrac{\pi n}{2},\; n \in \mathbb{Z}\).

Ответ: \(x = \dfrac{\pi}{12} + \dfrac{\pi n}{2},\; n \in \mathbb{Z}\).

Пример 2. Решить уравнение: \(\ctg^2 x - \ctg x - 2 = 0\). Найти корни на отрезке \([0, \pi]\).

Решение:

Замена: \(t = \ctg x\), \(t^2 - t - 2 = 0\).

\(t_1 = 2,\; t_2 = -1\).

\(\ctg x = 2 \Rightarrow x = \arcctg 2 + \pi n\).

\(\ctg x = -1 \Rightarrow x = \dfrac{3\pi}{4} + \pi n\).

На \([0, \pi]\): \(x = \arcctg 2,\; x = \dfrac{3\pi}{4}\).

Ответ: \(x = \arcctg 2,\; x = \dfrac{3\pi}{4}\).

📌 Запомните:
• Для \(\sin x = 1\) и \(\sin x = -1\) период \(2\pi\)
• Для \(\sin x = 0\) период \(\pi\)
• Для \(\cos x = 1\) и \(\cos x = -1\) период \(2\pi\)
• Для \(\cos x = 0\) период \(\pi\)
• \(n \in \mathbb{Z}\) (\(n\) — целое число)
