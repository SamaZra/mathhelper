Произведения

\(\sin\alpha\sin\beta = \dfrac{1}{2}[\cos(\alpha-\beta) - \cos(\alpha+\beta)]\)

\(\cos\alpha\cos\beta = \dfrac{1}{2}[\cos(\alpha-\beta) + \cos(\alpha+\beta)]\)

\(\sin\alpha\cos\beta = \dfrac{1}{2}[\sin(\alpha-\beta) + \sin(\alpha+\beta)]\)

📝 Примеры применения

Пример 1. Преобразовать в сумму: \(\sin 5\alpha\sin 3\alpha\).

Решение:

Используем формулу \(\sin\alpha\sin\beta = \dfrac{1}{2}[\cos(\alpha-\beta) - \cos(\alpha+\beta)]\):

\(\sin 5\alpha\sin 3\alpha = \dfrac{1}{2}[\cos(5\alpha-3\alpha) - \cos(5\alpha+3\alpha)] =\)

\(= \dfrac{1}{2}[\cos 2\alpha - \cos 8\alpha]\).

Ответ: \(\dfrac{1}{2}(\cos 2\alpha - \cos 8\alpha)\).

Пример 2. Преобразовать в сумму: \(\cos 40^\circ\cos 10^\circ\).

Решение:

Используем формулу \(\cos\alpha\cos\beta = \dfrac{1}{2}[\cos(\alpha-\beta) + \cos(\alpha+\beta)]\):

\(\cos 40^\circ\cos 10^\circ = \dfrac{1}{2}[\cos 30^\circ + \cos 50^\circ] =\)

\(= \dfrac{1}{2}\left[\dfrac{\sqrt{3}}{2} + \cos 50^\circ\right] = \dfrac{\sqrt{3}}{4} + \dfrac{\cos 50^\circ}{2}\).

Ответ: \(\dfrac{\sqrt{3}}{4} + \dfrac{\cos 50^\circ}{2}\).

Пример 3. Вычислить: \(\sin 75^\circ\cos 15^\circ\).

Решение:

Используем формулу \(\sin\alpha\cos\beta = \dfrac{1}{2}[\sin(\alpha-\beta) + \sin(\alpha+\beta)]\):

\(\sin 75^\circ\cos 15^\circ = \dfrac{1}{2}[\sin 60^\circ + \sin 90^\circ] =\)

\(= \dfrac{1}{2}\left[\dfrac{\sqrt{3}}{2} + 1\right] = \dfrac{\sqrt{3}}{4} + \dfrac{1}{2}\).

Ответ: \(\dfrac{\sqrt{3}}{4} + \dfrac{1}{2}\).
