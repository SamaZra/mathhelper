Возрастание и убывание функции

Функция возрастает на интервале, если для любых \(x_1 < x_2\) из этого интервала выполняется \(f(x_1) < f(x_2)\).

Функция убывает на интервале, если для любых \(x_1 < x_2\) из этого интервала выполняется \(f(x_1) > f(x_2)\).

Достаточное условие (связь с производной):

  • Если \(f'(x) > 0\) на интервале, то функция возрастает.
  • Если \(f'(x) < 0\) на интервале, то функция убывает.
Определение экстремумов

Точка максимума \(x_0\) — точка, в которой производная меняет знак с «+» на «−». Значение \(f(x_0)\) называется максимумом функции.

Точка минимума \(x_0\) — точка, в которой производная меняет знак с «−» на «+». Значение \(f(x_0)\) называется минимумом функции.

Точки максимума и минимума называются точками экстремума. Значения функции в этих точках — экстремумами.

Алгоритм нахождения экстремумов
  1. Найти производную \(f'(x)\).
  2. Найти критические точки: решить уравнение \(f'(x) = 0\).
  3. Определить знаки производной на интервалах, на которые критические точки разбивают область определения.
  4. Сделать вывод: если знак меняется с «+» на «−» — максимум, с «−» на «+» — минимум.
  5. Вычислить значения функции в точках экстремума.
Пример: найти экстремумы \(f(x) = x^3 - 3x\)
1. Найдём производную:
\(f'(x) = 3x^2 - 3\).

2. Найдём критические точки:
\(f'(x) = 0\)
\(3x^2 - 3 = 0\)
\(x^2 = 1\)
\(x = -1,\; x = 1\).

3. Определим знаки производной:
• На интервале \((-\infty, -1)\):
\(f'(-2) = 3(4) - 3 = 9 > 0\) — знак «+», функция возрастает.
• На интервале \((-1, 1)\):
\(f'(0) = -3 < 0\) — знак «−», функция убывает.
• На интервале \((1, +\infty)\):
\(f'(2) = 3(4) - 3 = 9 > 0\) — знак «+», функция возрастает.

4. Сделаем вывод о точках экстремума:
В точке \(x = -1\) знак меняется с «+» на «−» ⇒ точка максимума.
В точке \(x = 1\) знак меняется с «−» на «+» ⇒ точка минимума.

5. Найдём значения функции в точках экстремума:
\(f(-1) = (-1)^3 - 3(-1) = -1 + 3 = 2\) — максимум.
\(f(1) = 1^3 - 3\cdot 1 = 1 - 3 = -2\) — минимум.

Ответ: максимум в точке \((-1, 2)\), минимум в точке \((1, -2)\).
