📐 Площадь криволинейной трапеции

📐 Определение

Криволинейная трапеция — фигура, ограниченная графиком \(y = f(x)\) (где \(f(x) \ge 0\)), прямыми \(x = a\), \(x = b\) и осью \(Ox\).

\(S = \int_a^b f(x)\,dx\)
📋 Частные случаи
1. Фигура ограничена двумя функциями (между кривыми)
\(S = \int_a^b (f(x) - g(x))dx\),
где \(f(x) \ge g(x)\) на \([a,b]\)
2. Фигура состоит из нескольких частей
\(S = S_1 + S_2 + \dots =\)
\(= \int_a^c f(x)dx + \int_c^b g(x)dx + \dots\)
3. Фигура расположена под осью \(Ox\) (\(f(x) \le 0\))
\(S = -\int_a^b f(x)dx\)
📌 Пример с графиком
По графику найти площадь закрашенной фигуры:
На графике изображена парабола \(y = 4 - x^2\). Закрашенная область ограничена параболой и осью \(Ox\).
Шаг 1. По графику определяем пределы интегрирования
Парабола пересекает ось \(Ox\) в точках \(x = -2\) и \(x = 2\). Значит, \(a = -2\), \(b = 2\).
Шаг 2. По графику видим, что функция выше оси \(Ox\)
На отрезке \([-2, 2]\) парабола \(y = 4 - x^2 \ge 0\). Площадь равна интегралу от функции.
Шаг 3. Записываем интеграл
\(S = \int_{-2}^{2} (4 - x^2)\,dx\)
Шаг 4. Находим первообразную
\(F(x) = 4x - \frac{x^3}{3}\)
Шаг 5. Вычисляем по формуле Ньютона-Лейбница
\(F(2) = 4 \cdot 2 - \frac{8}{3} = 8 - \frac{8}{3} = \frac{16}{3}\)
\(F(-2) = 4 \cdot (-2) - \frac{(-2)^3}{3} = -8 + \frac{8}{3} = -\frac{16}{3}\)
\(S = F(2) - F(-2) = \frac{16}{3} - \left(-\frac{16}{3}\right) = \frac{32}{3}\)
Ответ: \(S = \frac{32}{3}\) кв. ед.
📌 Пример без графика
Найти площадь фигуры, ограниченной линиями:
\(y = x^2 - 3x\), \(y = 0\)
Шаг 1. Находим точки пересечения с осью \(Ox\)
\(x^2 - 3x = 0 \;\Rightarrow\; x(x - 3) = 0\)
\(\Rightarrow\; x_1 = 0,\; x_2 = 3\)
Пределы интегрирования: \(a = 0\), \(b = 3\).
Шаг 2. Определяем знак функции на отрезке \([0, 3]\)
При \(x = 1\): \(1^2 - 3 \cdot 1 = -2 < 0\).
Функция отрицательна.
Так как \(f(x) \le 0\), площадь равна интегралу со знаком минус.
Шаг 3. Записываем формулу площади
\(S = -\int_0^3 (x^2 - 3x)\,dx = \int_0^3 (3x - x^2)\,dx\)
Шаг 4. Находим первообразную
\(F(x) = \frac{3x^2}{2} - \frac{x^3}{3}\)
Шаг 5. Вычисляем по формуле Ньютона-Лейбница
\(F(3) = \frac{3 \cdot 9}{2} - \frac{27}{3} = \frac{27}{2} - 9 = 13,5 - 9 = 4,5\)
\(F(0) = 0\)
\(S = F(3) - F(0) = 4,5 - 0 = 4,5\)
Ответ: \(S = 4,5\) кв. ед.
📌 Пример: площадь между двумя кривыми
Найти площадь фигуры, ограниченной линиями:
\(y = x^2\), \(y = 2x\)
Шаг 1. Находим точки пересечения кривых
\(x^2 = 2x \;\Rightarrow\; x^2 - 2x = 0 \;\Rightarrow\; x(x - 2) = 0\)
\(\Rightarrow\; x_1 = 0,\; x_2 = 2\)
Пределы интегрирования: \(a = 0\), \(b = 2\).
Шаг 2. Определяем, какая функция выше на отрезке \([0, 2]\)
При \(x = 1\): \(y_1 = 1^2 = 1\), \(y_2 = 2 \cdot 1 = 2\).
Значит, \(2x \ge x^2\) на \([0, 2]\).
Верхняя функция: \(f(x) = 2x\), нижняя: \(g(x) = x^2\).
Шаг 3. Записываем формулу площади между кривыми
\(S = \int_a^b (f(x) - g(x))\,dx = \int_0^2 (2x - x^2)\,dx\)
Шаг 4. Находим первообразную
\(F(x) = 2 \cdot \frac{x^2}{2} - \frac{x^3}{3} = x^2 - \frac{x^3}{3}\)
Шаг 5. Вычисляем по формуле Ньютона-Лейбница
\(F(2) = 2^2 - \frac{2^3}{3} = 4 - \frac{8}{3} = \frac{12}{3} - \frac{8}{3} = \frac{4}{3}\)
\(F(0) = 0 - 0 = 0\)
\(S = F(2) - F(0) = \frac{4}{3} - 0 = \frac{4}{3}\)
Ответ: \(S = \frac{4}{3}\) кв. ед.

💡 Геометрический смысл:

Площадь фигуры, заключённой между параболой \(y = x^2\) и прямой \(y = 2x\) на отрезке \([0, 2]\).
