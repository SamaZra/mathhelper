Исследование функции с помощью производной

Алгоритм · Пояснения · Пример с графиком
Алгоритм исследования функции
  1. Найти область определения \(D(f)\).

    Исключаем точки, где знаменатель равен нулю, подкоренное выражение отрицательно.

  2. Найти производную \(f'(x)\).

    Используем таблицу производных и правила дифференцирования.

  3. Найти критические точки: \(f'(x) = 0\) или \(f'(x)\) не существует.

    Решаем уравнение, находим точки, принадлежащие области определения.

  4. Определить промежутки возрастания и убывания.

    Метод интервалов: если \(f'(x) > 0\) — возрастает, если \(f'(x) < 0\) — убывает.

  5. Найти точки экстремума и значения функции в них.

    Смена знака с «+» на «−» — максимум, с «−» на «+» — минимум.

  6. Найти точки пересечения с осями координат.

    С осью \(Oy\): \(x = 0\). С осью \(Ox\): \(f(x) = 0\).

  7. Построить график функции.

    На основе полученных данных строим эскиз графика.

Пример: исследовать \(f(x) = x^3 - 3x\)
1. Область определения: \(D(f) = \mathbb{R}\) (\(x\) — любое число).
2. Найдём производную, используя таблицу и правила:
\(f'(x) = (x^3)' - (3x)' = 3x^2 - 3\).
3. Приравняем производную к нулю и решим уравнение:
\(f'(x) = 0\)
\(3x^2 - 3 = 0\)
\(3(x^2 - 1) = 0\)
\(x^2 = 1\)
\(x = -1,\; x = 1\).
4. Метод интервалов:
Определим знаки производной и промежутки монотонности:
• На \((-\infty, -1)\): \(f'(x) > 0\) ⇒ функция возрастает.
• На \((-1, 1)\): \(f'(x) < 0\) ⇒ функция убывает.
• На \((1, +\infty)\): \(f'(x) > 0\) ⇒ функция возрастает.
5. Точки экстремума и координаты:
\(x = -1\): знак с «+» на «−» ⇒ максимум.
\(f(-1) = (-1)^3 - 3(-1) = -1 + 3 = 2\). Координаты: \((-1, 2)\).
\(x = 1\): знак с «−» на «+» ⇒ минимум.
\(f(1) = 1^3 - 3\cdot 1 = 1 - 3 = -2\). Координаты: \((1, -2)\).
6. Точки пересечения с осями:
С \(Oy\): \(f(0) = 0\) ⇒ \((0, 0)\).
С \(Ox\): \(x^3 - 3x = 0\)
\(x(x^2 - 3) = 0\)
\(x = 0,\; x = \pm\sqrt{3}\).
Координаты: \((-\sqrt{3}, 0)\), \((0, 0)\), \((\sqrt{3}, 0)\).
7. График функции:
График — кубическая парабола, проходит через \((-\sqrt{3}, 0)\), \((0,0)\), \((\sqrt{3}, 0)\), максимум \((-1, 2)\), минимум \((1, -2)\).
