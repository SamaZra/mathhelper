Логарифмические уравнения

📖 Что такое логарифмическое уравнение

Логарифмическое уравнение — уравнение, в котором переменная содержится под знаком логарифма.

$\log_2 x = 3$
$\log_3(x+1) = \log_3(2x-5)$
$\log_2^2 x - 3\log_2 x + 2 = 0$
$\lg(x^2-2x) = \lg(x+4)$
⚠️ Важно: При решении логарифмических уравнений обязательна проверка ОДЗ или подстановка в исходное уравнение! Под знаком логарифма может находиться только положительное число.
📐 Метод потенцирования $\log_a f(x) = \log_a g(x) \Rightarrow f(x) = g(x)$

Если логарифмы с одинаковым основанием равны, то равны и их аргументы.

Пример 1. $\log_3(x+1) = \log_3(2x-5)$

Шаг 1. ОДЗ. $x+1 > 0 \Rightarrow x > -1$
$2x-5 > 0 \Rightarrow x > 2{,}5$
Общая ОДЗ: $x > 2{,}5$.

Шаг 2. Потенцируем (убираем логарифмы). $x+1 = 2x-5$
$x = 6$.

Шаг 3. Проверка ОДЗ. $6 > 2{,}5$ — подходит.

Ответ: $x = 6$.
Пример 2. $\lg(x^2-2x) = \lg(x+4)$

Шаг 1. ОДЗ. $x^2-2x > 0 \Rightarrow x < 0$ или $x > 2$
$x+4 > 0 \Rightarrow x > -4$
Общая ОДЗ: $-4 < x < 0$ или $x > 2$.

Шаг 2. Потенцируем. $x^2-2x = x+4 \Rightarrow x^2-3x-4=0$.
Корни: $x_1 = -1$, $x_2 = 4$.

Шаг 3. Проверка ОДЗ. $x = -1$: $-4 < -1 < 0$ — подходит.
$x = 4$: $4 > 2$ — подходит.

Ответ: $x = -1$ или $x = 4$.
📐 Метод замены переменной сведение к квадратному уравнению
Пример 3. $\log_2^2 x - 3\log_2 x + 2 = 0$

Шаг 1. ОДЗ. $x > 0$.

Шаг 2. Замена $t = \log_2 x$. $t^2 - 3t + 2 = 0$.
Корни: $t_1 = 1$, $t_2 = 2$.

Шаг 3. Возврат. $\log_2 x = 1 \Rightarrow x = 2$
$\log_2 x = 2 \Rightarrow x = 4$
Оба подходят по ОДЗ.

Ответ: $x = 2$ или $x = 4$.
Пример 4. $\log_3^2 x + \log_3 x - 6 = 0$

Шаг 1. ОДЗ. $x > 0$. Замена $t = \log_3 x$.

Шаг 2. $t^2 + t - 6 = 0$. Корни: $t_1 = 2$, $t_2 = -3$.

Шаг 3. $\log_3 x = 2 \Rightarrow x = 9$. $\log_3 x = -3 \Rightarrow x = 3^{-3} = \dfrac{1}{27}$.
Оба подходят по ОДЗ.

Ответ: $x = \dfrac{1}{27}$ или $x = 9$.
📐 Метод приведения к одному основанию использование свойств логарифмов
Пример 5. $\log_2 x + \log_4 x = 6$

Шаг 1. ОДЗ. $x > 0$.

Шаг 2. Приводим к основанию $2$. $\log_4 x = \dfrac{\log_2 x}{\log_2 4} = \dfrac{\log_2 x}{2}$.
Уравнение: $\log_2 x + \dfrac{1}{2}\log_2 x = 6$, то есть $\dfrac{3}{2}\log_2 x = 6$.

Шаг 3. $\log_2 x = 4 \Rightarrow x = 16$.

Ответ: $x = 16$.
Пример 6. $\log_3 x - \log_9 x = 2$

Шаг 1. ОДЗ. $x > 0$.

Шаг 2. $\log_9 x = \dfrac{\log_3 x}{\log_3 9} = \dfrac{\log_3 x}{2}$.
$\log_3 x - \dfrac{1}{2}\log_3 x = 2 \Rightarrow \dfrac{1}{2}\log_3 x = 2$.

Шаг 3. $\log_3 x = 4 \Rightarrow x = 81$.

Ответ: $x = 81$.
📐 Логарифмирование показательных уравнений когда переменная и в основании, и в показателе
Пример 7. $x^{\log_2 x} = 16$

Шаг 1. ОДЗ. $x > 0$, $x \neq 1$.

Шаг 2. Логарифмируем по основанию $2$. $\log_2(x^{\log_2 x}) = \log_2 16$
$\log_2 x \cdot \log_2 x = 4$
$(\log_2 x)^2 = 4$.

Шаг 3. $\log_2 x = 2 \Rightarrow x = 4$, или $\log_2 x = -2 \Rightarrow x = \dfrac{1}{4}$.
Оба подходят по ОДЗ.

Ответ: $x = \dfrac{1}{4}$ или $x = 4$.
🎯 Задачи с параметром ★ логарифмические уравнения с параметром
Пример 8. При каких $a$ уравнение $\log_2(x+a) = \log_2(x-1)$ имеет решение?

Шаг 1. ОДЗ. $x+a > 0$, $x-1 > 0 \Rightarrow x > 1$, $x > -a$.

Шаг 2. Потенцируем. $x+a = x-1 \Rightarrow a = -1$.

Шаг 3. При $a = -1$ уравнение имеет вид $\log_2(x-1) = \log_2(x-1)$ — верно при любом $x > 1$.
Значит, решений бесконечно много при $a = -1$. При других $a$ решений нет.

Ответ: $a = -1$ (бесконечно много решений).
