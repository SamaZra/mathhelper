Иррациональные неравенства

📖 Что такое иррациональное неравенство

Иррациональное неравенство — неравенство, в котором переменная содержится под знаком корня (радикала).

$\sqrt{x+1} > 3$
$\sqrt{x+5} < x-1$
$\sqrt{x^2-4} \geq x+2$
$\sqrt{x} + \sqrt{x-1} \leq 3$
⚠️ Основной принцип: Иррациональные неравенства решаются методом равносильных переходов — переходом к системе (или совокупности) неравенств без корня с учётом ОДЗ.
📐 Корень меньше функции $\sqrt{f(x)} < g(x)$
$\sqrt{f(x)} < g(x) \Leftrightarrow \begin{cases} f(x) \geq 0 \\ g(x) > 0 \\ f(x) < g^2(x) \end{cases}$
Пример 1. $\sqrt{x+3} < x-1$

Шаг 1. Записываем равносильную систему. $\begin{cases} x+3 \geq 0 \\ x-1 > 0 \\ x+3 < (x-1)^2 \end{cases}$

Шаг 2. Решаем первые два неравенства. $x \geq -3$, $x > 1$. Итого: $x > 1$.

Шаг 3. Решаем третье неравенство. $x+3 < x^2-2x+1 \Rightarrow x^2-3x-2 > 0$.
Корни: $x = \dfrac{3 \pm \sqrt{17}}{2}$. Решение: $x < \dfrac{3-\sqrt{17}}{2}$ или $x > \dfrac{3+\sqrt{17}}{2}$.

Шаг 4. Пересекаем с $x > 1$. $x > \dfrac{3+\sqrt{17}}{2} \approx 3{,}56$.

Ответ: $x \in \left(\dfrac{3+\sqrt{17}}{2}; +\infty\right)$.
📐 Корень больше функции $\sqrt{f(x)} > g(x)$
$\sqrt{f(x)} > g(x) \Leftrightarrow \left[\begin{array}{l} \begin{cases} f(x) \geq 0 \\ g(x) < 0 \end{cases} \\ \begin{cases} g(x) \geq 0 \\ f(x) > g^2(x) \end{cases} \end{array}\right.$
Пример 2. $\sqrt{x^2-4} \geq x-2$

Шаг 1. Записываем равносильную совокупность. $\left[\begin{array}{l} \begin{cases} x^2-4 \geq 0 \\ x-2 < 0 \end{cases} \\ \begin{cases} x-2 \geq 0 \\ x^2-4 \geq (x-2)^2 \end{cases} \end{array}\right.$

Шаг 2. Первая система. $x \leq -2$ или $x \geq 2$, и $x < 2$. Пересечение: $x \leq -2$.

Шаг 3. Вторая система. $x \geq 2$, и $x^2-4 \geq x^2-4x+4 \Rightarrow 4x \geq 8 \Rightarrow x \geq 2$.
Пересечение: $x \geq 2$.

Шаг 4. Объединяем. $x \leq -2$ или $x \geq 2$.

Ответ: $x \in (-\infty; -2] \cup [2; +\infty)$.
📐 Нестрогое неравенство $\sqrt{f(x)} \geq g(x)$
$\sqrt{f(x)} \geq g(x) \Leftrightarrow \left[\begin{array}{l} \begin{cases} f(x) \geq 0 \\ g(x) \leq 0 \end{cases} \\ \begin{cases} g(x) \geq 0 \\ f(x) \geq g^2(x) \end{cases} \end{array}\right.$
Пример 3. $\sqrt{2x+1} \geq x-1$

Шаг 1. Записываем совокупность. $\left[\begin{array}{l} \begin{cases} 2x+1 \geq 0 \\ x-1 \leq 0 \end{cases} \\ \begin{cases} x-1 \geq 0 \\ 2x+1 \geq (x-1)^2 \end{cases} \end{array}\right.$

Шаг 2. Первая система. $x \geq -\dfrac{1}{2}$, $x \leq 1$. Пересечение: $-\dfrac{1}{2} \leq x \leq 1$.

Шаг 3. Вторая система. $x \geq 1$, и $2x+1 \geq x^2-2x+1 \Rightarrow x^2-4x \leq 0 \Rightarrow 0 \leq x \leq 4$.
Пересечение: $1 \leq x \leq 4$.

Шаг 4. Объединяем. $-\dfrac{1}{2} \leq x \leq 4$.

Ответ: $x \in \left[-\dfrac{1}{2}; 4\right]$.
📐 Неравенства с несколькими корнями уединение радикала и возведение в степень
Пример 4. $\sqrt{x+8} > \sqrt{x} + 2$

Шаг 1. ОДЗ. $x \geq 0$.

Шаг 2. Возводим в квадрат. $x+8 > x + 4\sqrt{x} + 4 \Rightarrow 4 > 4\sqrt{x} \Rightarrow \sqrt{x} < 1 \Rightarrow x < 1$.

Шаг 3. Учитываем ОДЗ. $0 \leq x < 1$.

Ответ: $x \in [0; 1)$.
📐 Неравенства с кубическими корнями корень нечётной степени — ОДЗ не нужен
Пример 5. $\sqrt[3]{x-2} > 1$

Шаг 1. ОДЗ. Корень кубический определён при любом $x$.

Шаг 2. Возводим в куб (знак сохраняется). $x-2 > 1 \Rightarrow x > 3$.

Ответ: $x \in (3; +\infty)$.
Пример 6. $\sqrt[3]{x+1} \leq \sqrt[3]{2x-3}$

Шаг 1. Возводим в куб: $x+1 \leq 2x-3 \Rightarrow x \geq 4$.

Ответ: $x \in [4; +\infty)$.
📐 Системы иррациональных неравенств пересечение решений
Пример 7. $\begin{cases} \sqrt{x-1} \leq 2 \\ \sqrt{5-x} > 1 \end{cases}$

Шаг 1. ОДЗ. $x \geq 1$, $x \leq 5$. Общая ОДЗ: $1 \leq x \leq 5$.

Шаг 2. Первое неравенство. $x-1 \leq 4 \Rightarrow x \leq 5$. С учётом ОДЗ: $1 \leq x \leq 5$.

Шаг 3. Второе неравенство. $5-x > 1 \Rightarrow x < 4$. С учётом ОДЗ: $1 \leq x < 4$.

Шаг 4. Пересекаем. $1 \leq x < 4$.

Ответ: $x \in [1; 4)$.
