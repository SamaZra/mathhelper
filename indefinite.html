?? Неопределённый интеграл

?? Определение

Неопределённый интеграл функции \(f(x)\) — это совокупность всех первообразных данной функции.

Обозначается \(\int f(x)\,dx = F(x) + C\), где \(F'(x) = f(x)\), а \(C\) — произвольная постоянная.

Символ \(\int\) (вытянутая буква S) ввёл Лейбниц, он происходит от латинского «summa».

?? Таблица интегралов
\(\int f(x)\,dx\)\(\int f(kx+b)\,dx\)
\(\int 1\,dx = x + C\)\(\int 7\,dx = 7x + C\)
\(\int x^n dx = \frac{x^{n+1}}{n+1} + C\)\(\int (kx+b)^n dx = \frac{(kx+b)^{n+1}}{k(n+1)} + C\)
\(\int \frac{1}{x} dx = \ln|x| + C\)\(\int \frac{dx}{kx+b} = \frac{1}{k} \ln|kx+b| + C\)
\(\int \sin x\,dx = -\cos x + C\)\(\int \sin(kx+b)\,dx = -\frac{\cos(kx+b)}{k} + C\)
\(\int \cos x\,dx = \sin x + C\)\(\int \cos(kx+b)\,dx = \frac{\sin(kx+b)}{k} + C\)
\(\int \frac{1}{\sin^2 x} dx = -\operatorname{ctg} x + C\)\(\int \frac{dx}{\sin^2(kx+b)} = -\frac{\operatorname{ctg}(kx+b)}{k} + C\)
\(\int \frac{1}{\cos^2 x} dx = \operatorname{tg} x + C\)\(\int \frac{dx}{\cos^2(kx+b)} = \frac{\operatorname{tg}(kx+b)}{k} + C\)
\(\int e^x dx = e^x + C\)\(\int e^{kx+b}\,dx = \frac{e^{kx+b}}{k} + C\)
\(\int a^x dx = \frac{a^x}{\ln a} + C\)\(\int a^{kx+b}\,dx = \frac{a^{kx+b}}{k\cdot\ln a} + C\)
?? Примеры для сложных функций
\(\int 7\,dx = 7x + C\)
\(\int (3x+2)^4\,dx = \frac{(3x+2)^5}{3\cdot 5} + C\)
\(\int \frac{dx}{4x-1} = \frac{1}{4} \ln|4x-1| + C\)
\(\int \sin(6x)\,dx = -\frac{\cos(6x)}{6} + C\)
\(\int \cos(2x+1)\,dx = \frac{\sin(2x+1)}{2} + C\)
\(\int \frac{dx}{\sin^2(4x)} = -\frac{\operatorname{ctg}(4x)}{4} + C\)
\(\int \frac{dx}{\cos^2(7x+\pi)} = \frac{\operatorname{tg}(7x+\pi)}{7} + C\)
\(\int e^{3x+2}\,dx = \frac{e^{3x+2}}{3} + C\)
\(\int 2^{5x}\,dx = \frac{2^{5x}}{5\cdot\ln 2} + C\)
?? Свойства неопределённого интеграла
Свойство 1 · интеграл суммы/разности
\(\int (f(x) \pm g(x))dx = \int f(x)dx \pm \int g(x)dx\)

\(\int (x^2 + \sin x)dx = \frac{x^3}{3} - \cos x + C\)

Свойство 2 · вынесение константы
\(\int a\cdot f(x)dx = a\cdot \int f(x)dx\)

\(\int 5\cos x\,dx = 5\sin x + C\)

Свойство 3 · линейная замена
\(\int f(kx+b)dx = \frac{1}{k}F(kx+b) + C\)

\(\int \sin(5x)dx = -\frac{\cos(5x)}{5} + C\)

Свойство 4 · интеграл от нуля
\(\int 0 \cdot dx = C\)

\(\int 0\,dx = C\) (константа)

Свойство 5 · интеграл от дифференциала
\(\int dx = x + C\)

\(\int 1\,dx = x + C\)

?? Метод непосредственного интегрирования

Пример 1. Найти интеграл: \(\int (3x^2 + 2x + 1)dx\)

Шаг 1. Разбиваем на сумму интегралов:

\(\int (3x^2 + 2x + 1)dx = \int 3x^2 dx + \int 2x dx + \int 1 dx\)

? Используем свойство 1: интеграл суммы равен сумме интегралов

Шаг 2. Выносим константы за знак интеграла:

\(= 3\int x^2 dx + 2\int x dx + \int 1 dx\)

? Используем свойство 2: \(\int a\cdot f(x)dx = a\cdot \int f(x)dx\)

Шаг 3. Применяем табличные формулы:

\(= 3\cdot\frac{x^3}{3} + 2\cdot\frac{x^2}{2} + x + C\)

? \(\int x^n dx = \frac{x^{n+1}}{n+1}\), \(\int 1 dx = x\)

Шаг 4. Упрощаем:

\(= x^3 + x^2 + x + C\)

Ответ: \(x^3 + x^2 + x + C\)

Пример 2. Найти интеграл: \(\int \frac{x^4 + 2x^3 - x}{x}dx\)

Шаг 1. Упрощаем подынтегральное выражение:

\(\int \frac{x^4 + 2x^3 - x}{x}dx = \int \left(\frac{x^4}{x} + \frac{2x^3}{x} - \frac{x}{x}\right)dx\)

? Почленно делим числитель на знаменатель

Шаг 2. Сокращаем дроби:

\(= \int (x^3 + 2x^2 - 1)dx\)

? \(\frac{x^4}{x}=x^3\), \(\frac{2x^3}{x}=2x^2\), \(\frac{x}{x}=1\)

Шаг 3. Интегрируем почленно:

\(= \int x^3 dx + 2\int x^2 dx - \int 1 dx\)

? Разбиваем на сумму интегралов и выносим константу 2

Шаг 4. Применяем табличные формулы:

\(= \frac{x^4}{4} + 2\cdot\frac{x^3}{3} - x + C\)

? \(\int x^3 dx = \frac{x^4}{4}\), \(\int x^2 dx = \frac{x^3}{3}\), \(\int 1 dx = x\)

Ответ: \(\frac{x^4}{4} + \frac{2x^3}{3} - x + C\)

Пример 3. Найти интеграл: \(\int 3x^2(2x - 4)dx\)

Шаг 1. Раскрываем скобки:

\(\int 3x^2(2x - 4)dx = \int (6x^3 - 12x^2)dx\)

? Умножаем: \(3x^2\cdot 2x = 6x^3\), \(3x^2\cdot(-4) = -12x^2\)

Шаг 2. Интегрируем почленно:

\(= 6\int x^3 dx - 12\int x^2 dx\)

? Выносим константы за знак интеграла

Шаг 3. Применяем табличные формулы:

\(= 6\cdot\frac{x^4}{4} - 12\cdot\frac{x^3}{3} + C\)

? \(\int x^3 dx = \frac{x^4}{4}\), \(\int x^2 dx = \frac{x^3}{3}\)

Шаг 4. Упрощаем:

\(= \frac{6x^4}{4} - 4x^3 + C = \frac{3x^4}{2} - 4x^3 + C\)

? Сокращаем: \(\frac{6}{4}=\frac{3}{2}\), \(\frac{12}{3}=4\)

Ответ: \(\frac{3x^4}{2} - 4x^3 + C\)

?? Частые ошибки

? Забывают прибавить константу \(C\) — неопределённый интеграл = семейство функций, а не одна функция.

? Неправильно делят на коэффициент \(k\) — при интегрировании \(f(kx+b)\) всегда делим на \(k\): \(\int \sin(3x)dx = -\frac{\cos(3x)}{3} + C\).

? Путают знаки в тригонометрических интегралах — \(\int \sin x = -\cos x\), а \(\int \cos x = \sin x\).

? Проверяйте себя производной: \((F(x) + C)'\) должно равняться подынтегральной функции \(f(x)\).
