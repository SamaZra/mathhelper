# Математический справочник (MathHelper)

Универсальный справочник по математике для школьников, студентов и преподавателей.

## Что внутри

- **Алгебра**: ФСУ, степени, корни, логарифмы, показательная функция
- **Тригонометрия**: тождества, формулы, таблица значений, уравнения
- **Геометрия**: стереометрия с 3D-моделями, многогранники, тела вращения, векторы
- **Анализ**: производные, интегралы, площадь криволинейной трапеции

## Инструменты

- Калькулятор (обычный, инженерный, программистский)
- Калькулятор производных и интегралов
- Калькулятор радикалов и квадратных уравнений
- Конвертер величин (валюта, длина, масса, площадь, температура, ИМТ)
- Построитель графиков функций

## Особенности

- 🔍 Полнотекстовый поиск по всем темам
- ⭐ Избранное — сохраняйте нужные страницы
- 🌙 Тёмная и светлая тема
- 🧊 3D-модели с настройкой качества
- 📝 Тесты с таймером для самопроверки
- 📴 Работает офлайн

## Для кого

Приложение создано действующим преподавателем математики. Подходит для подготовки к ЕГЭ, ОГЭ, экзаменам и повседневной учёбы.
