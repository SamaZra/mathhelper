Тригонометрические неравенства

📖 Что такое тригонометрическое неравенство

Тригонометрическое неравенство — неравенство, содержащее переменную под знаком тригонометрической функции.

$\sin x > \dfrac{1}{2}$
$\cos x \leq -\dfrac{\sqrt{2}}{2}$
$\operatorname{tg} x < 1$
$\sin 2x \geq \dfrac{\sqrt{3}}{2}$
💡 Ключевой принцип: Решение тригонометрических неравенств основано на единичной окружности и периодичности функций. Решения записываются в виде промежутков с учётом периода.
📐 $\sin x > a$ и $\sin x < a$ решение с использованием единичной окружности
Пример 1. $\sin x > \dfrac{1}{2}$

Шаг 1. На единичной окружности отмечаем точки, где $\sin x = \dfrac{1}{2}$. $x_1 = \dfrac{\pi}{6}$, $x_2 = \dfrac{5\pi}{6}$.

Шаг 2. Выбираем дугу, где $\sin x > \dfrac{1}{2}$ (выше прямой $y = \dfrac{1}{2}$). Это промежуток $\left(\dfrac{\pi}{6}; \dfrac{5\pi}{6}\right)$.

Шаг 3. Добавляем период $2\pi n$. Ответ: $x \in \left(\dfrac{\pi}{6} + 2\pi n; \dfrac{5\pi}{6} + 2\pi n\right)$, $n \in \mathbb{Z}$.
Пример 2. $\sin x \leq -\dfrac{\sqrt{2}}{2}$

Шаг 1. Точки: $x_1 = \dfrac{5\pi}{4}$, $x_2 = \dfrac{7\pi}{4}$.

Шаг 2. Выбираем дугу, где $\sin x \leq -\dfrac{\sqrt{2}}{2}$ (ниже прямой $y = -\dfrac{\sqrt{2}}{2}$). Это промежуток $\left[\dfrac{5\pi}{4}; \dfrac{7\pi}{4}\right]$.

Ответ: $x \in \left[\dfrac{5\pi}{4} + 2\pi n; \dfrac{7\pi}{4} + 2\pi n\right]$, $n \in \mathbb{Z}$.
📐 $\cos x > a$ и $\cos x < a$ решение с использованием единичной окружности
Пример 3. $\cos x < -\dfrac{1}{2}$

Шаг 1. Точки: $\cos x = -\dfrac{1}{2} \Rightarrow x_1 = \dfrac{2\pi}{3}$, $x_2 = \dfrac{4\pi}{3}$.

Шаг 2. Выбираем дугу, где $\cos x < -\dfrac{1}{2}$. Это промежуток $\left(\dfrac{2\pi}{3}; \dfrac{4\pi}{3}\right)$.

Ответ: $x \in \left(\dfrac{2\pi}{3} + 2\pi n; \dfrac{4\pi}{3} + 2\pi n\right)$, $n \in \mathbb{Z}$.
Пример 4. $\cos x \geq \dfrac{\sqrt{3}}{2}$

Шаг 1. Точки: $x_1 = \dfrac{\pi}{6}$, $x_2 = -\dfrac{\pi}{6}$ (или $\dfrac{11\pi}{6}$).

Шаг 2. Выбираем дугу: $\left[-\dfrac{\pi}{6}; \dfrac{\pi}{6}\right]$.

Ответ: $x \in \left[-\dfrac{\pi}{6} + 2\pi n; \dfrac{\pi}{6} + 2\pi n\right]$, $n \in \mathbb{Z}$.
📐 $\operatorname{tg} x > a$ и $\operatorname{tg} x < a$ использование линии тангенсов

Решение неравенств с тангенсом удобно выполнять с помощью линии тангенсов (касательная к окружности в точке $(1;0)$).

Пример 5. $\operatorname{tg} x \leq 1$

Шаг 1. На линии тангенсов отмечаем точку $1$. $\operatorname{tg} x = 1 \Rightarrow x = \dfrac{\pi}{4} + \pi n$.

Шаг 2. Выбираем промежуток. $x \in \left(-\dfrac{\pi}{2} + \pi n; \dfrac{\pi}{4} + \pi n\right]$.

Ответ: $x \in \left(-\dfrac{\pi}{2} + \pi n; \dfrac{\pi}{4} + \pi n\right]$, $n \in \mathbb{Z}$.
Пример 6. $\operatorname{tg} x > -\sqrt{3}$

Шаг 1. $\operatorname{tg} x = -\sqrt{3} \Rightarrow x = -\dfrac{\pi}{3} + \pi n$.

Шаг 2. $x \in \left(-\dfrac{\pi}{3} + \pi n; \dfrac{\pi}{2} + \pi n\right)$.

Ответ: $x \in \left(-\dfrac{\pi}{3} + \pi n; \dfrac{\pi}{2} + \pi n\right)$, $n \in \mathbb{Z}$.
📐 Неравенства с преобразованиями сведение к простейшим неравенствам
Пример 7. $\sin 2x > \dfrac{\sqrt{3}}{2}$

Шаг 1. Вводим замену $t = 2x$. $\sin t > \dfrac{\sqrt{3}}{2} \Rightarrow t \in \left(\dfrac{\pi}{3} + 2\pi n; \dfrac{2\pi}{3} + 2\pi n\right)$.

Шаг 2. Возврат. $2x \in \left(\dfrac{\pi}{3} + 2\pi n; \dfrac{2\pi}{3} + 2\pi n\right)$.
$x \in \left(\dfrac{\pi}{6} + \pi n; \dfrac{\pi}{3} + \pi n\right)$.

Ответ: $x \in \left(\dfrac{\pi}{6} + \pi n; \dfrac{\pi}{3} + \pi n\right)$, $n \in \mathbb{Z}$.
Пример 8. $\cos\left(x - \dfrac{\pi}{4}\right) \leq \dfrac{\sqrt{2}}{2}$

Шаг 1. Замена $t = x - \dfrac{\pi}{4}$. $\cos t \leq \dfrac{\sqrt{2}}{2} \Rightarrow t \in \left[\dfrac{\pi}{4} + 2\pi n; \dfrac{7\pi}{4} + 2\pi n\right]$.

Шаг 2. Возврат. $x - \dfrac{\pi}{4} \in \left[\dfrac{\pi}{4} + 2\pi n; \dfrac{7\pi}{4} + 2\pi n\right]$.
$x \in \left[\dfrac{\pi}{2} + 2\pi n; 2\pi + 2\pi n\right]$.

Ответ: $x \in \left[\dfrac{\pi}{2} + 2\pi n; 2\pi + 2\pi n\right]$, $n \in \mathbb{Z}$.
📊 Сводная таблица
$\sin x > a$ ($-1 < a < 1$): $x \in (\arcsin a + 2\pi n; \pi - \arcsin a + 2\pi n)$

$\sin x < a$ ($-1 < a < 1$): $x \in (-\pi - \arcsin a + 2\pi n; \arcsin a + 2\pi n)$

$\cos x > a$ ($-1 < a < 1$): $x \in (-\arccos a + 2\pi n; \arccos a + 2\pi n)$

$\cos x < a$ ($-1 < a < 1$): $x \in (\arccos a + 2\pi n; 2\pi - \arccos a + 2\pi n)$

$\operatorname{tg} x > a$: $x \in (\operatorname{arctg} a + \pi n; \dfrac{\pi}{2} + \pi n)$

$\operatorname{tg} x < a$: $x \in (-\dfrac{\pi}{2} + \pi n; \operatorname{arctg} a + \pi n)$

Важно: Для нестрогих неравенств ($\geq$, $\leq$) промежутки становятся замкнутыми. Для $\sin x > 1$ и $\sin x < -1$ решений нет.
