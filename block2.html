Декартовы координаты в пространстве

?? Прямоугольная система координат

В пространстве вводится прямоугольная (декартова) система координат $Oxyz$, состоящая из:

  • начала координат — точки $O$;
  • трёх взаимно перпендикулярных осей: $Ox$ (ось абсцисс), $Oy$ (ось ординат), $Oz$ (ось аппликат);
  • единичных отрезков на каждой оси.
Любая точка $A$ в пространстве имеет координаты $A(x, y, z)$.
Координаты вектора $\overrightarrow{AB}$: если $A(x_1, y_1, z_1)$, $B(x_2, y_2, z_2)$, то

$\overrightarrow{AB}\{x_2 - x_1,\; y_2 - y_1,\; z_2 - z_1\}$.
?? Пример:
$P(2, -1, 3)$, $Q(5, 2, -1)$:
$\overrightarrow{PQ}\{3,\; 3,\; -4\}$
$|\overrightarrow{PQ}| = \sqrt{9+9+16} = \sqrt{34}$
?? Координатные векторы и разложение по ним

Координатные векторы (орты): $\vec{i}\{1,0,0\}$, $\vec{j}\{0,1,0\}$, $\vec{k}\{0,0,1\}$.

Разложение:

$\vec{a} = x\vec{i} + y\vec{j} + z\vec{k}$

Разложение по трём некомпланарным векторам: любой вектор $\vec{a}$ можно единственным образом разложить по трём некомпланарным векторам $\vec{p}$, $\vec{q}$, $\vec{r}$:

$\vec{a} = \alpha\vec{p} + \beta\vec{q} + \gamma\vec{r}$

где $\alpha$, $\beta$, $\gamma$ — единственные числа (коэффициенты разложения).

?? Пример:
$\vec{a}\{3, -2, 5\}$ ? $\vec{a} = 3\vec{i} - 2\vec{j} + 5\vec{k}$
?? Компланарные векторы

Компланарные векторы — лежат в одной плоскости или параллельных плоскостях.

  • Любые два вектора всегда компланарны.
  • Если $\vec{c} = \alpha\vec{a} + \beta\vec{b}$, то они компланарны.
?? Пример: Проверим, являются ли векторы $\vec{r}\{1, 1, 0\}$, $\vec{p}\{1, 0, 0\}$ и $\vec{q}\{0, 1, 0\}$ компланарными.

Шаг 1. Проверяем, можно ли выразить $\vec{r}$ через $\vec{p}$ и $\vec{q}$:
$\vec{r} = \alpha\vec{p} + \beta\vec{q}$
$\{1, 1, 0\} = \alpha\{1, 0, 0\} + \beta\{0, 1, 0\}$

Шаг 2. Составляем систему уравнений по координатам:
По $x$: $1 = \alpha \cdot 1 + \beta \cdot 0 \;\Rightarrow\; \alpha = 1$
По $y$: $1 = \alpha \cdot 0 + \beta \cdot 1 \;\Rightarrow\; \beta = 1$
По $z$: $0 = \alpha \cdot 0 + \beta \cdot 0 \;\Rightarrow\; 0 = 0$ (верно)

Шаг 3. Нашли коэффициенты $\alpha = 1$, $\beta = 1$.
Подставляем: $\vec{r} = 1 \cdot \vec{p} + 1 \cdot \vec{q}$
$\{1, 1, 0\} = \{1, 0, 0\} + \{0, 1, 0\}$
Равенство верно.

Ответ: векторы компланарны, так как $\vec{r}$ выражается через $\vec{p}$ и $\vec{q}$.
?? Длина вектора и расстояние
Модуль:

$|\vec{a}| = \sqrt{x^2 + y^2 + z^2}$
Расстояние:

$d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2 + (z_2-z_1)^2}$
?? Пример: Найти расстояние между точками $A(1, 2, 3)$ и $B(4, 6, 8)$.

Шаг 1. Записываем формулу и подставляем координаты:
$d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2 + (z_2-z_1)^2}$
$x_2 - x_1 = 4 - 1 = 3$
$y_2 - y_1 = 6 - 2 = 4$
$z_2 - z_1 = 8 - 3 = 5$

Шаг 2. Вычисляем:
$d = \sqrt{3^2 + 4^2 + 5^2} = \sqrt{9 + 16 + 25} = \sqrt{50}$

Шаг 3. Упрощаем:
$d = \sqrt{50} = \sqrt{25 \cdot 2} = 5\sqrt{2}$

Ответ: $d = 5\sqrt{2}$
