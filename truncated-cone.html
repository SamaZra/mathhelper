⏢ Усечённый конус

📐 3D-модель прямого усечённого конуса
🔵 Нижний радиус (R) 🔷 Верхний радиус (r) 🟢 Высота (h) 🟠 Образующая (L) 🟡 Осевое сечение (трапеция)

Интерактивная модель: вращайте пальцем, масштабируйте двумя пальцами (щипок). Нажмите ⛶ для полноэкранного режима.

📖 Определение
  • Прямой усечённый конус — тело, образованное вращением прямоугольной трапеции вокруг боковой стороны, перпендикулярной основаниям.
  • Основания — два круга разных радиусов (\(R\) и \(r\)).
  • Боковая поверхность — часть конической поверхности.
🔷 Элементы
  • Нижний радиус \(R\), верхний радиус \(r\)
  • Высота (\(h\)), образующая (\(\ell\))
  • \(\ell = \sqrt{h^{2} + (R - r)^{2}}\)
✂️ Сечения
  • Сечение параллельно основаниям — круг.
  • Осевое сечение — равнобедренная трапеция.
  • Наклонное сечение — часть эллипса.
🧮 Формулы
\[ S_{\text{бок}} = \pi \ell (R + r) \]
\[ S_{\text{полн}} = S_{\text{бок}} + \pi R^{2} + \pi r^{2} \]
\[ V = \frac{1}{3}\pi h (R^{2} + Rr + r^{2}) \]
📋 Свойства
  • Основания — два круга в параллельных плоскостях.
  • Образующая \(\ell = \sqrt{h^{2} + (R - r)^{2}}\).
  • Развёртка — часть кругового кольца.
  • Все образующие равны.
📌 Запомните:
• Образующая: \(\ell = \sqrt{h^{2} + (R - r)^{2}}\)
• Объём: \(V = \frac{1}{3}\pi h (R^{2} + Rr + r^{2})\)
