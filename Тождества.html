Основные тождества

\(\sin^2 \alpha + \cos^2 \alpha = 1\)

\(\tg \alpha = \dfrac{\sin \alpha}{\cos \alpha}\)

\(\ctg \alpha = \dfrac{\cos \alpha}{\sin \alpha}\)

\(\tg \alpha \cdot \ctg \alpha = 1\)

\(1 + \tg^2 \alpha = \dfrac{1}{\cos^2 \alpha}\)

\(1 + \ctg^2 \alpha = \dfrac{1}{\sin^2 \alpha}\)

📝 Примеры применения тождеств

Пример 1. Упростить выражение: \(\sin^2\alpha \cdot \ctg^2\alpha + \sin^2\alpha\).

Решение:

Используем тождество \(\ctg\alpha = \dfrac{\cos\alpha}{\sin\alpha}\):

\(\sin^2\alpha \cdot \dfrac{\cos^2\alpha}{\sin^2\alpha} + \sin^2\alpha = \cos^2\alpha + \sin^2\alpha\).

Используем основное тождество \(\sin^2\alpha + \cos^2\alpha = 1\):

\(\cos^2\alpha + \sin^2\alpha = 1\).

Ответ: 1.

Пример 2. Дано: \(\tg\alpha = 2\). Найти \(\cos\alpha\).

Решение:

Используем тождество \(1 + \tg^2\alpha = \dfrac{1}{\cos^2\alpha}\):

\(1 + 2^2 = \dfrac{1}{\cos^2\alpha}\)

\(5 = \dfrac{1}{\cos^2\alpha}\)

\(\cos^2\alpha = \dfrac{1}{5} = 0,2\)

\(\cos\alpha = \pm\sqrt{0,2} = \pm\dfrac{\sqrt{5}}{5}\).

Ответ: \(\cos\alpha = \pm\dfrac{\sqrt{5}}{5}\) (знак зависит от четверти).

Пример 3. Дано: \(\sin\alpha = 0,6\). Найти \(\cos\alpha\) и \(\tg\alpha\), если \(\alpha \in (0^\circ, 90^\circ)\).

Решение:

Используем тождество \(\sin^2\alpha + \cos^2\alpha = 1\):

\(\cos^2\alpha = 1 - \sin^2\alpha = 1 - 0,36 = 0,64\)

\(\cos\alpha = \sqrt{0,64} = 0,8\) (в первой четверти косинус положителен).

Используем тождество \(\tg\alpha = \dfrac{\sin\alpha}{\cos\alpha}\):

\(\tg\alpha = \dfrac{0,6}{0,8} = 0,75\).

Ответ: \(\cos\alpha = 0,8,\; \tg\alpha = 0,75\).

💡 Важно: Знак тригонометрической функции (sin, cos, tg, ctg) зависит от четверти, в которой находится угол.
