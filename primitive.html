`n

Первообразная

F'(x) = f(x) · множество функций + C
?? Определение

Первообразная функции \(f(x)\) — такая функция \(F(x)\), что \(F'(x) = f(x)\).

Если \(F(x)\) — первообразная, то \(F(x) + C\) — тоже первообразная (общее решение).

+C означает семейство функций, сдвинутых по вертикали.

?? Таблица первообразных

\(1\)\(x + C\)

\(\frac{1}{x}\)\(\ln|x| + C\)
\(\sin x\)\(-\cos x + C\)
\(\cos x\)\(\sin x + C\)
\(\frac{1}{\sin^2 x}\)\(-\operatorname{ctg} x + C\)
\(\frac{1}{\cos^2 x}\)\(\operatorname{tg} x + C\)
\(e^x\)\(e^x + C\)
\(a^x\)\(\frac{a^x}{\ln a} + C\)
?? Примеры
\(f(x)=5 \;\Rightarrow\; F(x)=5x + C\)
\(f(x)=x^4 \;\Rightarrow\; F(x)=\frac{x^5}{5} + C\)
\(f(x)=\frac{2}{x} \;\Rightarrow\; F(x)=2\ln|x| + C\)
\(f(x)=2\sin x \;\Rightarrow\; F(x)=-2\cos x + C\)
\(f(x)=3\cos x \;\Rightarrow\; F(x)=3\sin x + C\)
\(f(x)=\frac{4}{\sin^2 x} \;\Rightarrow\; F(x)=-4\operatorname{ctg} x + C\)
\(f(x)=\frac{5}{\cos^2 x} \;\Rightarrow\; F(x)=5\operatorname{tg} x + C\)
\(f(x)=3e^x \;\Rightarrow\; F(x)=3e^x + C\)
\(f(x)=4\cdot 2^x \;\Rightarrow\; F(x)=4\cdot\frac{2^x}{\ln 2} + C\)
?? Правила
Правило 1 · сумма / разность
\(F(x) \pm G(x)\) — первообразная для \(f(x) \pm g(x)\)
\(f(x)=x^2+\sin x\)
\(F(x)=\frac{x^3}{3}-\cos x + C\)
Правило 2 · постоянный множитель
\(a\cdot F(x)\) — первообразная для \(a\cdot f(x)\)
\(f(x)=5\cos x\)
\(F(x)=5\sin x + C\)
Правило 3 · сложная функция
\(\frac{1}{k}F(kx+b)\) — первообразная для \(f(kx+b)\)
\(f(x)=\sin(5x)\)
\(F(x)=-\frac{\cos(5x)}{5} + C\)
