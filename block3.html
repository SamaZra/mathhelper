Простейшие задачи в координатах

📌 Задача 1: Координаты середины отрезка
Если $A(x_1, y_1, z_1)$ и $B(x_2, y_2, z_2)$, то середина $M$ отрезка $AB$:

$$M\left( \frac{x_1 + x_2}{2},\; \frac{y_1 + y_2}{2},\; \frac{z_1 + z_2}{2} \right)$$

Обратная задача: если известны координаты середины $M$ и одного из концов отрезка, то координаты другого конца находятся по формулам:

$x_2 = 2x_M - x_1$
$y_2 = 2y_M - y_1$
$z_2 = 2z_M - z_1$

📐 Пример: Даны точки $A(3, -2, 5)$ и $B(7, 4, -3)$. Найти координаты середины отрезка $AB$.

$x_M = \dfrac{3+7}{2}=5$
$y_M = \dfrac{-2+4}{2}=1$
$z_M = \dfrac{5+(-3)}{2}=1$

Ответ: $M(5,1,1)$
📏 Задача 2: Длина вектора по его координатам
Для вектора $\vec{a}\{x, y, z\}$:

$|\vec{a}| = \sqrt{x^2 + y^2 + z^2}$

📐 Пример: Дан вектор $\vec{a}\{3, -4, 12\}$. Найти его длину.

$|\vec{a}| = \sqrt{3^2+(-4)^2+12^2}=$
$= \sqrt{9+16+144}= \sqrt{169}=13$

Для вектора $\overrightarrow{AB}$:
$\overrightarrow{AB} = \{x_2-x_1,\; y_2-y_1,\; z_2-z_1\}$
$|\overrightarrow{AB}| = \sqrt{(x_2-x_1)^2+(y_2-y_1)^2+(z_2-z_1)^2}$

📍 Задача 3: Расстояние между двумя точками
$d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2 + (z_2-z_1)^2}$

📐 Пример: Найти расстояние между $A(1,2,3)$ и $B(4,6,8)$.

$d = \sqrt{(4-1)^2+(6-2)^2+(8-3)^2}=$
$= \sqrt{3^2+4^2+5^2}= \sqrt{50}=5\sqrt{2}$

🧩 Ещё пример: $P(2,-1,3)$, $Q(5,2,-1)$:

$d = \sqrt{(5-2)^2+(2-(-1))^2+(-1-3)^2}=$
$= \sqrt{3^2+3^2+(-4)^2}= \sqrt{34}$
🎯 Задача 4: Деление отрезка в заданном отношении ★

Если точка $M$ делит отрезок $AB$ в отношении $\lambda = \dfrac{AM}{MB}$, то:

$M\left( \dfrac{x_1 + \lambda x_2}{1 + \lambda},\; \dfrac{y_1 + \lambda y_2}{1 + \lambda},\; \dfrac{z_1 + \lambda z_2}{1 + \lambda} \right)$

При $\lambda = 1$ точка $M$ — середина отрезка.

📐 Пример: Даны точки $A(1,2,3)$ и $B(7,8,9)$. Найти точку $M$, которая делит отрезок $AB$ в отношении $2:1$ ($\lambda = 2$).

$x_M = \dfrac{1 + 2 \cdot 7}{1 + 2} = \dfrac{15}{3} = 5$
$y_M = \dfrac{2 + 2 \cdot 8}{1 + 2} = \dfrac{18}{3} = 6$
$z_M = \dfrac{3 + 2 \cdot 9}{1 + 2} = \dfrac{21}{3} = 7$

Ответ: $M(5,6,7)$
🔺 Задача 5: Центр масс треугольника ★

Координаты центра масс (точки пересечения медиан) треугольника с вершинами $A(x_1,y_1,z_1)$, $B(x_2,y_2,z_2)$, $C(x_3,y_3,z_3)$:

$G\left( \dfrac{x_1+x_2+x_3}{3},\; \dfrac{y_1+y_2+y_3}{3},\; \dfrac{z_1+z_2+z_3}{3} \right)$

📐 Пример: Дан треугольник с вершинами $A(1,2,3)$, $B(4,5,6)$, $C(7,8,9)$. Найти центр масс.

$x_G = \dfrac{1+4+7}{3} = 4$
$y_G = \dfrac{2+5+8}{3} = 5$
$z_G = \dfrac{3+6+9}{3} = 6$

Ответ: $G(4,5,6)$
📌 Шпаргалка:

Середина: $M\left(\dfrac{x_1+x_2}{2},\dfrac{y_1+y_2}{2},\dfrac{z_1+z_2}{2}\right)$
Расстояние: $d = \sqrt{(x_2-x_1)^2+(y_2-y_1)^2+(z_2-z_1)^2}$
Деление в отношении $\lambda$: $M\left(\dfrac{x_1+\lambda x_2}{1+\lambda},\dfrac{y_1+\lambda y_2}{1+\lambda},\dfrac{z_1+\lambda z_2}{1+\lambda}\right)$
Центр масс: $G\left(\dfrac{x_1+x_2+x_3}{3},\dfrac{y_1+y_2+y_3}{3},\dfrac{z_1+z_2+z_3}{3}\right)$
